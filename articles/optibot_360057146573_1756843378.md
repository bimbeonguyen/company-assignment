[L1] # Advanced: How to Manually Install & Activate OptiSigns Remote Agent on Linux or Raspberry Pi

[L2] 

[L3] ### In this article, we will show you how to complete a manual installation of the OptiSigns Remote Agent on a Linux or Raspberry Pi device.

[L4] 

[L5] If you're running OptiSigns on a Windows, Linux or Raspberry Pi device, you can install optisigns-remote-agent add-on that will allow you to remotely control your device from the portal.

[L6] 

[L7] If you have not tried to automatically activate this feature, please try it first by following [this guide](https://support.optisigns.com/hc/en-us/articles/360055486554).

[L8] 

[L9] If automatic activation fails, follow these steps to manually install & activate the OptiSigns remote agent on your Linux or Raspberry Pi device.

[L10] 

[L11] **Before you start, please ensure that:**

[L12] 

[L13] * Your device has OptiSigns Digital Signage player installed. Download the latest version [here](https://www.optisigns.com/download).

[L14] * Your device has an active network connection.

[L15] * You have the authorization and privileges to install apps and background services on your device.

[L16] 

[L17] |  |

[L18] | --- |

[L19] | **NOTE** |

[L20] | The Remote Control feature is only available to subscribers with a **Pro Plus** **plan or above.** |

[L22] ## **How to manually install optisigns-remote-agent**

[L23] 

[L24] Open Terminal.

[L25] 

[L26] On Raspberry Pi run:

[L27] 

[L28] ```

[L29] curl -s https://release.optisigns.com/optisigns-remote-agent-setup-rpi.sh -L | sh

[L30] ```

[L31] 

[L32] On Linux run:

[L33] 

[L34] ```

[L35] curl -s https://release.optisigns.com/optisigns-remote-agent-setup-linux.sh -L | sh

[L36] ```

[L37] 

[L38] This will download a script and run installation.

[L39] 

[L40] ![mceclip0.png](https://support.optisigns.com/hc/article_attachments/360095986553)

[L41] 

[L42] After you see "Starting Remote Control Service ..." you are done!

[L43] 

[L44] You can go back to app.optisigns.com to use Remote Control feature now.

[L50] ## **More Technical Details for optisigns-remote-agent**

[L51] 

[L52] The binary executable is downloaded and installed to ~/bin/optisigns-remote-agent

[L53] 

[L54] It will be started as a background service and will automatically boot on system start up.

[L55] 

[L56] This ensures the remote agent is independent from OptiSigns Digital Signage Player and still running in case the Player encounters an issue, or you need to reinstall it.

[L57] 

[L58] You can manually execute the optisigns-remote-agent executable:

[L59] 

[L60] ```

[L61] <path>/optisigns-remote-agent [arg]  

[L62] install : to install remote agent  

[L63] remove : to remove/uninstall remote agent  

[L64] start or restart : to start remote agent service  

[L65] stop : to stop remote agent service  

[L66] version or -V : to print remote agent version  

[L67] help : to print this instructions  

[L68] run : to run the remote agent in console or terminal or service

[L69] ```

[L70] 

[L71] ## **How to remove optisigns-remote-agent:**

[L72] 

[L73] Open Terminal and run:

[L74] 

[L75] ```

[L76] ~/bin/optisigns-remote-agent remove

[L77] ```
