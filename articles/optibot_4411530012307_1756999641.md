[L1] # How to use Microsoft Teams Live app with OptiSigns

[L2] 

[L3] You can broadcast a Microsoft Teams Live on your digital signs with OptiSigns.

[L4] 

[L5] Sign in to [Office 365](https://www.office.com/) using your credentials.

[L6] 

[L7] Go to Teams:¬†

[L8] 

[L9] ![mceclip3.png](https://support.optisigns.com/hc/article_attachments/4411529735443)

[L10] 

[L11] Click on Calendar:

[L12] 

[L13] ![mceclip4.png](https://support.optisigns.com/hc/article_attachments/4411539049107)

[L14] 

[L15] Under New meeting, select Live event (*It is better to download Microsoft Teams Desktop app for running the live event):*

[L16] 

[L17] ![mceclip5.png](https://support.optisigns.com/hc/article_attachments/4411539067667)

[L18] 

[L19] Give the event a name and location, then click Next:

[L20] 

[L21] *![mceclip10.png](https://support.optisigns.com/hc/article_attachments/4411539271187)*

[L22] 

[L23] Make sure to set it as public, no login is required:

[L24] 

[L25] ![mceclip11.png](https://support.optisigns.com/hc/article_attachments/4411539357971)Click Schedule, then 'Get attendee link' to copy attendee URL:

[L26] 

[L27] ![mceclip12.png](https://support.optisigns.com/hc/article_attachments/4411539406483)¬†

[L28] 

[L29] Log in to OptiSigns:  

[L30] Click [Files/Assets -> Apps](https://app.optisigns.com/app/assetManagement) -> and select Microsoft Teams Live:

[L31] 

[L32] ![mceclip6.png](https://support.optisigns.com/hc/article_attachments/4411539156371)

[L33] 

[L34] Give your MS Teams Live app a name and paste the copied attendee URL in there:

[L35] 

[L36] ![mceclip13.png](https://support.optisigns.com/hc/article_attachments/4411539484051)

[L37] 

[L38] For not Public Live event (Sign-in required):

[L39] 

[L40] ![mceclip16.png](https://support.optisigns.com/hc/article_attachments/4411521581715)

[L41] 

[L42] You can set Username and Password in Advanced section:

[L43] 

[L44] ![mceclip14.png](https://support.optisigns.com/hc/article_attachments/4411521456787)

[L45] 

[L46] ¬†

[L47] 

[L48] **Important Note:** If you are utilizing the OptiSigns Android Stick kindly ensure that the Live Meeting is set to **public**. If your preference is to have the Teams Live Meeting set to "**Sign-in required**," it is recommended to consider using the Windows or Linux device.

[L49] 

[L50] ¬†

[L51] 

[L52] That's it!

[L53] 

[L54] Now you can assign this MS Teams Live app to your screens, or put it on a schedule so the screens will join automatically when the event start.

[L55] 

[L56] **Tips:**

[L57] 

[L58] Have the host start the event a few mins before screens are joining automatically.  

[L59] It will take a device 10-15s to join the event, so it's not a good idea to put the MS Teams Live in a playlist. It's better to use it as single asset to take up full screen or be part of a [SplitScreen zone](https://support.optisigns.com/hc/en-us/articles/360026559573).

[L60]
