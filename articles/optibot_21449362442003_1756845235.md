[L2]

[L5] ![Pro Player.png](https://support.optisigns.com/hc/article_attachments/38310905084947)

[L6] 

[L7] ## OptiSigns Pro Player Gen 2 Commands

[L8] 

[L9] You can install/update OptiSigns player on your OptiSigns Pro Player with 1 command line script:

[L10] 

[L11] ```

[L12]  ~/bin/manual_update.sh 

[L13] ```

[L14] 

[L15] This script will download latest version of OptiSigns AppImage and run it.

[L16] 

[L17] You can use our Remote Access feature to access the OptiSigns Pro Player, go to this article & copy the command line and run on the Pi to update. It will save time vs. manually clicking & downloading methods.

[L18] 

[L19]  

[L20] 

[L21] Other command line scripts:

[L22] 

[L23] * If your TV has color saturation issues, run this command to fix it:  

[L24] 

[L25] ```

[L26] wget -O- <https://download.optisignsapp.com/pro-player/change-color-profile.sh> | bash 

[L27] ```

[L28] * Overwrite the automatic security updates timing

[L29] 

[L30] ```

[L31] bash~/bin/override_upgrade_schedule.sh 23:30

[L32] ```

[L33] * To lock the USB port, if you want to lock the device to prevent tampering fully.

[L34] 

[L35] ```

[L36] ~/bin/optisigns-lockusb.sh

[L37] ```

[L38] * To unlock the USB port.

[L39] 

[L40] ```

[L41] ~/bin/optisigns-unlockusb.sh

[L42] ```
