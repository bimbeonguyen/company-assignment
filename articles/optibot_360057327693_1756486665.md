[L1] # Advanced: How to Install & Activate OptiSigns Remote Agent on Windows

[L2] 

[L3] ### In this article, we will explain how to install the OptiSigns Remote Agent on a Windows advice, which allows you to remotely control your device via the OptiSigns Portal.

[L4] 

[L5] If you are running OptiSigns on a Windows, Linux or Raspberry Pi device, you can install the optisigns-remote-agent add-on that will allow you to remotely control your device from the portal.

[L6] 

[L7] Before we start, please make sure you have the following:

[L8] 

[L9] * Your device has the OptiSigns Digital Signage player installed. Download the latest version [here](https://www.optisigns.com/download).

[L10] * Your device has a network connection.

[L11] * You have the authorization, privilege to install apps and background services on your Windows device

[L12] * Windows version 7 or later.

[L13] 

[L14] |  |

[L15] | --- |

[L16] | **NOTE** |

[L17] | The Remote Control feature is only available to subscribers with a **Pro Plus** **plan or above.** |

[L18] 

[L19] ## **How to Install & Activate OptiSigns Remote Agent on Windows**

[L20] 

[L21] Download [OptiSigns Remote Agent for Windows](https://links.optisigns.com/a83m).

[L22] 

[L23] Run it, and follow the steps to install.

[L24] 

[L25] ![mceclip0.png](https://support.optisigns.com/hc/article_attachments/360093737774)

[L26] 

[L27] Once installation is finished, go to the OptiSigns portal at [app.optisigns.com](https://app.optisigns.com/)

[L28] 

[L29] Click **Edit Screen ŌåÆ** **Advanced ŌåÆ More** to expand device management options.  

[L30] Then click **Activate.**

[L31] 

[L32] ![](https://support.optisigns.com/hc/article_attachments/19187687767443)

[L33] 

[L34] It will take a moment to activate (15-30 seconds) to download & install the agent, depending on your internet connection. After successful activation, click **Open** to begin a remote session with your device.

[L35] 

[L36] ![](https://support.optisigns.com/hc/article_attachments/19187761012883)

[L37] 

[L38] That's it, your remote session will be open in a new browser tab for you to use.

[L39]
