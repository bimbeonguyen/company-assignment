[L1] # How to Use the YouTube Dashboard App

[L2] 

[L3] ### In this article, we'll show how to display YouTube Dashboard information on your digital signs with OptiSigns.

[L4] 

[L5] * [What You'll Need](#WhatYouNeed)

[L6] * [Accessing Looker Studio and Linking YouTube Dashboard Data](#AccessingLooker)

[L7] * [Generating Your Dashboard from a Template](#GeneratingDashboard)

[L8] * [Creating a YouTube Dashboard App](#CreatingApp)

[L9] * [Deploying a YouTube Dashboard App](#DeployingApp)

[L10] 

[L11] With the YouTube Dashboard app, it's possible to display near real-time dashboards of information pulled straight from YouTube Dashboard. These dashboards are created in Google Looker Studio and can be customized, then communicated throughout the company via digital signage.

[L12] 

[L13] Let's get started.

[L14] 

[L15] ---

[L16] 

[L17] ## What You'll Need

[L18] 

[L19] * An OptiSigns account - [**Standard Plan** or higher](https://www.optisigns.com/pricing)

[L20] * A YouTube Studio account, set up and enabled

[L21] * Access to Looker Studio

[L22] * An [OptiSigns-enabled device](https://support.optisigns.com/hc/en-us/articles/360021855653-What-hardware-and-devices-are-supported)

[L23] * A screen, [set up and paired with OptiSigns](https://support.optisigns.com/hc/en-us/articles/18823504383891-OptiSigns-Getting-Started-Guide)

[L24] 

[L25] ---

[L26] 

[L27] ## Accessing Looker Studio and Linking YouTube Analytics Data

[L28] 

[L29] In order to get your YouTube Analytics data into the proper format, you'll need to export it to Looker Studio. Start by going to [lookerstudio.google.com](https://lookerstudio.google.com) and logging in with your Google Account.

[L30] 

[L31] Now, connect your account as a data source by clicking **Create** then **Data source**:

[L32] 

[L33] ![](https://support.optisigns.com/hc/article_attachments/48626106655379)

[L34] 

[L35] Now click **YouTube Analytics** as your Connector.

[L36] 

[L37] ![](https://support.optisigns.com/hc/article_attachments/48626115789843)

[L38] 

[L39] You'll need to **Authorize** Looker Studio to use your YouTube Channel, set the **Account**, then hit **Connect.**

[L40] 

[L41] ![](https://support.optisigns.com/hc/article_attachments/48626115793043)

[L42] 

[L43] Once connected, you'll be able to set your data, including which fields and parameters you'd like to include in Looker Studio. Select whichever you'd like to display.

[L44] 

[L45] From here, you have two options:

[L46] 

[L47] 1. Click **Create Report**. This will allow you to create and customize your YouTube Analytics report from scratch. Do this if you want granular control over how your data looks.

[L48] 2. Use a **YouTube Channel** template. This will allow you to quickly organize your data. Use this if you want your data on screen quickly.

[L49] 

[L50] This guide will assume you choose to use a Template. If you choose Create Report, feel free to skip to the next section.

[L51] 

[L52] ---

[L53] 

[L54] ## Generating Your Dashboard from a Template

[L55] 

[L56] Back out to the Looker Studio homepage, then select **YouTube Channel Report**.

[L57] 

[L58] ![](https://support.optisigns.com/hc/article_attachments/48626106662035)

[L59] 

[L60] This will generate a sample report. Click **Use My Own Data:**

[L61] 

[L62] ![](https://support.optisigns.com/hc/article_attachments/48626106663315)

[L63] 

[L64] Now, choose the **Account,** then hit **Add.**

[L65] 

[L66] If done correctly, you'll see a green dot in the Use my own data area.

[L67] 

[L68] Now, set up the report the way you like it. Then, hit **Share**, and make sure the report is set to **Public:**

[L69] 

[L70] ![](https://support.optisigns.com/hc/article_attachments/48656734362515)

[L71] 

[L72] This allows it to be set up, and is necessary for it to display.

[L73] 

[L74] ---

[L75] 

[L76] ## Creating Your YouTube Dashboard App

[L77] 

[L78] Once you have your report set up the way you'd like to display it, click **Edit**:

[L79] 

[L80] ![](https://support.optisigns.com/hc/article_attachments/48626115798675)

[L81] 

[L82] Now go to **File ŌåÆ Embed report**.

[L83] 

[L84] ![](https://support.optisigns.com/hc/article_attachments/48626106668691)

[L85] 

[L86] Now you should be on the **Embed Report** screen. Click **Enable embedding**, then make sure it is displaying an Embed Code. Change your **Width** and **Height** to match the resolution of the display you plan to use as a digital sign - we generally recommend 1920x1080 if you're unsure - then click **Copy to Clipboard.**

[L87] 

[L88] ![](https://support.optisigns.com/hc/article_attachments/48626115803027)

[L89] 

[L90] Next, go to the OptiSigns portal. Go to **Assets ŌåÆ Add Asset ŌåÆ Apps.**

[L91] 

[L92] ![](https://support.optisigns.com/hc/article_attachments/48626106670611)

[L93] 

[L94] Select **YouTube Dashboard:**

[L95] 

[L96] ![](https://support.optisigns.com/hc/article_attachments/48626115806995)

[L97] 

[L98] Now you can set up your YouTube Dashboard app:

[L99] 

[L100] ![](https://support.optisigns.com/hc/article_attachments/48626106675603)

[L101] 

[L102] * **Name -** The name of your YouTube Dashboard asset. This is for organizational purposes within OptiSigns only and will not display on your screens.

[L103] * **Embed Code** **-** The Embed Code you retrieved a moment ago should be pasted into this field.

[L104] * **Update Interval -** How often OptiSigns will ping your Embed URL to update the data displayed on it. This is measured in seconds.

[L105] 

[L106] |  |

[L107] | --- |

[L108] | **NOTE** |

[L109] | The Preview at this stage may display oddly. This is ok. We recommend pushing your YouTube Dashboard app to a real screen for proper testing. Afterward, we can come back to this and alter the Width and Height elements if they need adjusting. |

[L110] 

[L111] Now, hit **Save**. You'll have created a YouTube Dashboard app.

[L115] ## Deploying a YouTube Dashboard App

[L116] 

[L117] You can deploy your new YouTube Dashboard app as an individual asset, or as part of a [Split Screen](https://support.optisigns.com/hc/en-us/articles/360026559573-How-to-Create-and-Use-the-Split-Screen-App).

[L118] 

[L119] To get your new YouTube Dashboard asset to a screen, go to the **Screens** tab, then click the screen you want to assign it to.

[L120] 

[L121] ![](https://support.optisigns.com/hc/article_attachments/48626106679315)

[L122] 

[L123] This brings up the **Edit Screen** tab:

[L124] 

[L125] ![](https://support.optisigns.com/hc/article_attachments/48656464012819)

[L126] 

[L127] Here, select **Asset** under Content type, then hit **Change** next to Selected Asset.

[L128] 

[L129] Then, select your created Google Analytics Asset:

[L130] 

[L131] ![](https://support.optisigns.com/hc/article_attachments/48626115811347)

[L132] 

[L133] Now hit **Save**. Your Google Analytics asset will now display on screen.┬Ā

[L134] 

[L135] |  |

[L136] | --- |

[L137] | **NOTE** |

[L138] | You may need to alter the Width and Height on the Embed Code to get it to display exactly the way you want it. Be sure to push it to the intended screen first to test it - the Preview sometimes looks odd. |

[L139] 

[L140] You can also deploy it as part of a split screen, allowing you to show other assets at the same time. See how in our [Split Screen app article.](https://support.optisigns.com/hc/en-us/articles/360026559573-How-to-Create-and-Use-the-Split-Screen-App) It can also be displayed in a Playlist or Schedule.
