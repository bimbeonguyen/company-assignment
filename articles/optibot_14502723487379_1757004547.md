[L1] # How to use AeriCast Add-on for Wireless Presentation and Video Conferencing

[L2] 

[L3] With the AeriCast add-on, you can turn any conference room TV into a digital sign. Aericast also allows user to present from their laptops to those screens without needing cables, dongles or extra hardware.

[L4] 

[L5] With a USB webcam, mic and speakers connected to the OptiSigns Windows Player, users can wirelessly link these devices and use them in the video conference.

[L6] 

[L7] It's done in three easy steps:

[L8] 

[L9] 1. Create AeriCast Wireless Presentation app in your "Files/Assets"

[L10] 2. Assign it to your screen

[L11] 3. Wirelessly mirror your screen from a [laptop](https://support.optisigns.com/hc/en-us/articles/14502378549907) or [iOS/Android phone, tablet](https://support.optisigns.com/hc/en-us/articles/14557706459283), or begin [AeriCast Add-on wireless conferencing](https://support.optisigns.com/hc/en-us/articles/23030445716883/)

[L12] 

[L13] **Note:**  

[L14] All customers can use AeriCast add-on with a 14-day free trial.

[L15] 

[L16] ![aericast add-on trial activation](https://support.optisigns.com/hc/article_attachments/42852986048915)

[L17] 

[L18] The trial allows two free licenses of the AeriCast add-on. When the trial is over, each created AeriCast add-on will cost $20 per month.

[L30] ## **Setting Up the Aericast Add-On:**

[L31] 

[L32] ### 1. Create AeriCast Wireless Presentation app

[L33] 

[L34] First, you will need to have your [screens set up and paired](https://www.optisigns.com/blog/how-to-set-up-digital-signs-with-optisigns-and-amazon-fire-tv).

[L35] 

[L36] Then log on to our portal:┬Ā<http://app.optisigns.com/>

[L37] 

[L38] Go to **Files/Assets ŌåÆ** **"App"** and search for the **"AeriCast Wireless Presentation"** app.

[L39] 

[L40] ![](https://support.optisigns.com/hc/article_attachments/14502480771987)

[L41] 

[L42] Enter your AeriCast Wireless Presentation app information:

[L43] 

[L44] ![](https://support.optisigns.com/hc/article_attachments/14502481770131)

[L45] 

[L46] * **Name:** Name of your AeriCast Wireless Presentation app, this is the name of the app in your asset list. It will not┬Ābe displayed on your screens.

[L47] * **Title:** Enter a title to be displayed on your screen. For example: "Welcome to Conf. Room N123"

[L48] * **Message:** Enter a message to be displayed on your screen. This could be anything: a fun workplace meme, a reminder message, or instructions on how to use the room.

[L49] * **Expires in:** The 4 digit presenter codes are set to expire every 10 minutes by default. This is to prevent someone saving the code and using it days or weeks later.

[L50] * **Background Playlist:** By default, the app plays a few images in the background. This is for aesthetics, and also to prevent your screen from developing stuck pixels. You can customize what images play here with a Playlist. Note: only images will be played.

[L51] * **Background Opacity:** This appears only when a Playlist is selected and allows the user to change the opacity of the background so that the text still stands out effectively.

[L52] * **Text Color:** Default is white, but if you'd like another color, this option provides them.

[L53] * **Different Presenter Code for each screens**: When you assign any device, the pairing code will be different on different devices.

[L54] 

[L55] |  |

[L56] | --- |

[L57] | **NOTE:** The **Different Presenter Code for Each Screens** option is only supported on the PAID Aericast subscription. If you want to show the same Presenter Code across multiple screens, it can only be done with a paid subscription. |

[L58] 

[L59] Click **"Advanced"** to set:

[L60] 

[L61] ![firefox_rThQm0OTta.png](https://support.optisigns.com/hc/article_attachments/31248390996243)

[L62] 

[L63] * **Instruction 1:**The default message is "To present," this can be changed to whatever you like.

[L64] * **Instruction 2****:** The default message is "present.optisigns.com," this can be changed to whatever you like.

[L65] * **Instruction 3:** The default message is "and enter the code below to start presenting," this can be changed to whatever you like.

[L66] * **Scrolling strip:** By default is "To present, go to present.optisigns.com and enter", this can be changed to whatever you like.

[L67] 

[L68] Click **Save**.  

[L69] After saving, you can **P****review** the wall. It may take several seconds for the app to load.
