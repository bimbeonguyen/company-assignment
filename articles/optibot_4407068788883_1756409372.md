[L1] # Map View & Screen Location settings

[L2] 

[L3] You can view your devices on a map.

[L4] 

[L5] This helps to visualize your deployment, and select certain screens based on geo locations to make changes to them.

[L6] 

[L7] You can Drag select or select all visible screens on the map.

[L8] 

[L9] ![Map View of screens](https://support.optisigns.com/hc/article_attachments/4407075918099)

[L10] 

[L11]  

[L12] 

[L13] For the screen to be shown up on the map, you will need to set its location.

[L14] 

[L15] To set the location, open Edit the screen.

[L16] 

[L17] Click Advance and search/type in the location.

[L18] 

[L19] Screen location will also be used for apps that need location such as Weather.
