[L1] # How to enable SSH for your Raspberry Pi

[L2] 

[L3] By default SSH is disabled on Raspberry Pi (for both if you setting your own or bought the RPi from us).

[L4] 

[L5] To Enable SSH:

[L6] 

[L7] Open "Raspberry Pi Configuration"

[L8] 

[L9] ![mceclip0.png](https://support.optisigns.com/hc/article_attachments/1500011889561)

[L10] 

[L11] Select "Interfaces" tab and change SSH to Enable

[L12] 

[L13] ![mceclip1.png](https://support.optisigns.com/hc/article_attachments/1500011889661)

[L14] 

[L15] Go back to "System" and Change your password to something secured and you know.  

[L16] If you create your own SD Card, default Raspberry Pi password is "raspberry" and it's not secured at all.  

[L17] If you buy devices from us, you need to change the password to something else to use SSH.

[L18] 

[L19] ![mceclip2.png](https://support.optisigns.com/hc/article_attachments/1500011611442)

[L20] 

[L21]  

[L22] 

[L23] After that you can SSH into your device.

[L24] 

[L25] You can read more about SSH here:

[L26] 

[L27] <https://www.raspberrypi.org/documentation/remote-access/ssh/>

[L28] 

[L29]  

[L30] 

[L31] If you have further questions feel free to contact our support team.
