[L1] # Install/Update OptiSigns player on RPi via script

[L2] 

[L3] You can install/update OptiSigns player on your Raspberry Pi with 1 command line script:

[L4] 

[L5] ```

[L6] curl -s https://links.optisigns.com/rpi-update-sh -L | sh

[L7] ```

[L8] 

[L9] This script will download latest version of OptiSigns AppImage and run it.

[L10] 

[L11] You can use our Remote Access feature to access the RPi, go to this article & copy the command line and run on the Pi to update. It will save time vs. manually clicking & downloading method.
