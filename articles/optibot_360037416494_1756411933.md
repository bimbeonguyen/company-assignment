[L1] # How to use the Atlassian JIRA app

[L2] 

[L3] ### Jira is a popular project management tool that helps teams plan, track, release, and support work. Let's go through how to set this app integration up!

[L4] 

[L5] **What you'll need:**

[L6] 

[L7] * Jira project or dashboard

[L8] * Keyboard and mouse

[L9] 

[L10] ---

[L11] 

[L12] In your Jira account, navigate to the page of the project or private dashboard that you'd like to display. Copy the direct URL.

[L13] 

[L14] ![Jira dashboard with the URL pointed at](https://support.optisigns.com/hc/article_attachments/35838410787219)

[L15] 

[L16] Next, log in to [OptiSigns portal](https://app.optisigns.com/app/screenManagement) and navigate to the files/assets page. Click Apps and search for Jira.¬†

[L17] 

[L18] ![Jira app within OptiSigns](https://support.optisigns.com/hc/article_attachments/35838410792595)

[L19] 

[L20] Here, you'll input certain information to create the asset:

[L21] 

[L22] ![Jira app settings when setting it up in OptiSigns](https://support.optisigns.com/hc/article_attachments/35838425274899)

[L23] 

[L24] * **Name:** Any name you'd like to give the asset for internal organizational purposes only.

[L25] * **Type:** Either URL or Embed code.

[L26] * **URL/Embed Code:** The direct link or code of the project or dashboard that you'd like to display.

[L27] * **Update Interval:** The duration of time between updates. (Default: 600 seconds).

[L28] 

[L29] For "Sign In" settings, you will have to provide you log in so OptiSigns can easily log in for you. When your app is pushed to the screen, OptiSigns will use this information to automatically login to your account to display the URL/Embed code given:

[L30] 

[L31] * **Master Password:** By default, OptiSigns will encrypt the whole script with OptiSigns private key to protect your script, especially username, password in the script. You can add another protection layer by entering a Master Password. If you enter Master Password here, at each device, you will need to enter that Master Password one time in OptiSigns app so it can decrypt the content.

[L32] * **Username/Email:** The email or username for the account you're using.

[L33] * **Password:** The password for the account you're using.

[L34] 

[L35] Finally, click "**Save**".

[L36] 

[L37] ### Verification Code

[L38] 

[L39] When you push your Jira asset to your screen for the first time, Jira will send a verification code to your email. You will need to input this code into the asset pushed to your screen.

[L40] 

[L41] For this, you will need a **keyboard and mouse**connected to your device to input this. Afterward, Jira will recognize your device and not ask you to do this again.

[L42] 

[L43] [According to Jira](https://community.atlassian.com/t5/Jira-questions/Stop-asking-me-for-email-verification/qaq-p/1646087), this is for security purposes so Jira can register the device to your account.¬†

[L44]

[L47] ## Troubleshooting

[L48] 

[L49] * If your dashboard is public, it may display as empty. Making your dashboard private will allow OptiSigns to properly run the login script.
