[L18] ## 

[L19]  **Edit your Playlist**

[L20] 

[L21] To edit a playlist, simply **click** on the playlist.

[L22] 

[L23] Modify attributes, add, delete, or rearrange items, and the content on your screens will automatically update.

[L24] 

[L25] **1.** To edit all items in the playlist, click **Select All** button in the top left corner.

[L26] 

[L27] or, hover over the item in the playlist and check the checkbox.

[L28] 

[L29] **2.** Select assets within your playlist to access **Remove from Playlist** and **Edit Duration**┬Āoptions.

[L30] 

[L31] |  |

[L32] | --- |

[L33] | **Remove from Playlist:** This option will delete selected asset(s) from the playlist. |

[L34] | **Edit Duration:** You will be able to customize the duration in seconds for all asset(s). |

[L35] 

[L36] ## 

[L37]  **Add Additional Items Into a Playlist**

[L38] 

[L39] To add items from your files/asset to your playlist:

[L40] 

[L41] **1.** Select an asset from the right panel, or hold the Ctrl key to select multiple items.

[L42] 

[L43] **2.** Click the **'Add Selected Items to Playlist'** button OR just simply Drag and Drop to the Playlist.

[L44] 

[L45] * This action will promptly incorporate the chosen assets into your preferred playlist.

[L51] ## 

[L52] **Playlist Options and Change Multiple Playlists Settings at Once**

[L53] 

[L54] #### **Singular Playlist Options:**

[L55] 

[L56] **1.** Click the **gear icon** on the top right-hand side.![](https://support.optisigns.com/hc/article_attachments/28565848186003)┬Ā

[L57] **2.** The Playlist Option page will then appear:![](https://support.optisigns.com/hc/article_attachments/28565863777555)┬Ā

[L58] **3.** The following Playlist Option features will be listed below.* **Scale Image:** The option will default to Fit. This option will allow you to resize your Image per the selected preference:

[L59] 	+ None

[L60] 	+ Fit

[L61] 	+ Stretch

[L62] 	+ Zoom

[L63] * **Scale Video:** The option will default to Fit. This option will allow you to resize your Video per the selected preference:

[L64] 	+ None

[L65] 	+ Fit

[L66] 	+ Stretch

[L67] 	+ Zoom

[L68] * **Scale Document:** The option will default to Fit. This option will allow you to resize your Document per the selected preference:

[L69] 	+ None

[L70] 	+ Fit

[L71] 	+ Stretch

[L72] 	+ Zoom

[L73] * **Shuffle:** The "shuffle" option for a playlist randomizes the playback order of items.

[L74] * **Default Transition:** This option determines the type of transition effect that occurs when switching between items in the playlist:

[L75] 	+ Fade

[L76] 	+ Slide from Right

[L77] 	+ Slide from Left

[L78] 	+ Rotate In

[L79] 	+ Flip

[L80] 	+ Gallery

[L81] 	+ None

[L82] * **Transition Speed:** This option controls the duration or speed at which the transition effect occurs when switching between items in the playlist:

[L83] 	+ Slow

[L84] 	+ Medium

[L85] 	+ Fast

[L86] 	+ Custom: You will have the option to select a specific duration in seconds, with choices ranging from 1 to 20 in five-second intervals.

[L87] 

[L88] **4.** Click the Advanced button:

[L89] 

[L90] ┬Ā┬Ā* **Playlist - Max Duration:** The maximum duration for your playlist playback in seconds. The minimum duration you can set this to is 4 seconds.

[L91] * **Adv. playback settings apply to:** Apply restrictions on the playlist items so that the item will only play on certain screens or not play on certain screens:

[L92] 	+ All

[L93] 	+ Play on

[L94] 	+ Not play on

[L95] * **Target Screens:** This field will only be visible if you have selected the option "**Play on**" or "**Not play on**" in the "**Adv. playback settings apply to**" option.

[L96] 	+ Select the target screens by either Screen name or Screen Tags.

[L97] * **Screen Tags**: Add a tag name.

[L98] * **Screens:**┬ĀChoose your paired screen.

[L99] * **Target tags:** A grouping method for screens, enabling you to push content to tags, with all screens assigned to those tags receiving the content.

[L100] 	+ **Use Asset Default:** By mixing content for different screens within the same playlist, you can simplify the management of your screens.

[L101] 		- You will also need to set up content tagging which you can learn more about it [here](https://support.optisigns.com/hc/en-us/articles/20879903340947-How-to-Use-The-Asset-Tag-Features-in-The-Playlist).

[L102] 	+ **Use Playlist Default:**┬ĀBy mixing content for different screens within the same playlist, you can simplify the management of your screens.

[L103] 		- Create a tag name.

[L104] 		- You will also need to set up content tagging which you can learn more about [here](https://support.optisigns.com/hc/en-us/articles/20879903340947-How-to-Use-The-Asset-Tag-Features-in-The-Playlist).

[L105] 		- Make sure to select "Use Playlist Default".

[L106] 		- Under the "Content Tag Rule" under Edit Screens simply select the playlist tag instead of the asset tag.

[L107] * **Item Schedule:** Control the schedule of each playlist item.

[L108] * **Sub-playlist control:** With Sub-playlist control, you can define how many items you want to play from the sub-playlist and if it is embedded as a nested playlist. It can help you better manage the time allocation of a certain type of content.

[L109] * **Resume on next play:** With this option enabled, the playlist will resume from where it left off last time, rather than starting from the beginning. This can enhance the user experience, especially when the playlist is interrupted for any reason.

[L110] * **Respect Orientation:** Choose the orientation you would like for your playlist:

[L111] 	+ None

[L112] 	+ Both

[L113] 	+ Landscape

[L114] 	+ Portrait

[L115] * **Background Music:** You will be able to add music to your playlist here. You'll have to upload an MP3 file to OptiSigns before you can select this.
