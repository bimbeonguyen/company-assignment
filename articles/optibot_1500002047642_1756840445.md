[L1] # Advanced: How to assign security for users to only edit certain zones of the screens

[L2] 

[L3] This article will guide you through how to assign security for users to only edit certain zones of the screens.  

[L4] Use Case: a Company with 2 departments, Department A and Department B, each will have a section/zone of the screen that they can update freely, but they cannot change the zone belong to other departments. Administrators can edit modify the whole screen.

[L5] 

[L6] ![mceclip0.png](https://support.optisigns.com/hc/article_attachments/360102952113)

[L7] 

[L8] ┬Ā

[L9] 

[L10] To do this we will need to:

[L11] 

[L12] 1) You need to add the users to the Account Member. Please set them as a User. Here is a related [article](https://support.optisigns.com/hc/en-us/articles/360016990233-Multi-users-Invite-your-team-members-to-your-account).

[L13] 

[L14] ![mceclip22.png](https://support.optisigns.com/hc/article_attachments/360102942833)

[L15] 

[L16] 2) Create Files/Assets folders for Department A & Department B

[L17] 

[L18] ![mceclip23.png](https://support.optisigns.com/hc/article_attachments/360100781194)

[L19] 

[L20] 3) Create 2 playlists: Department A & Department B

[L21] 

[L22] ![mceclip13.png](https://support.optisigns.com/hc/article_attachments/360102942173)

[L23] 

[L24] 4) Create a SplitScreen & Assign playlist

[L25] 

[L26] ![mceclip4.png](https://support.optisigns.com/hc/article_attachments/1500001932161)

[L27] 

[L28] 5) Set up security for Screens, Files/Assets, Playlist folders

[L29] 

[L30] For the screen: You can edit the "Change Permission" in the folder. You can set **Admin Only**.

[L31] 

[L32] Put your screens in this folder so only Admins can edit & change assignment for the screens.

[L33] 

[L34] ![mceclip5.png](https://support.optisigns.com/hc/article_attachments/1500001932181)

[L35] 

[L36] ![mceclip21.png](https://support.optisigns.com/hc/article_attachments/360100781154)

[L37] 

[L38] For the Department A folder in Files/Assets: You can edit the "Change Permission" in the folder. You can set it to **DepartmentA User**.

[L39] 

[L40] ![mceclip10.png](https://support.optisigns.com/hc/article_attachments/360102941973)

[L41] 

[L42] For the Department B folder in Files/Assets: You can edit the "Change Permission" in the folder. You can set it to **DepartmentB User**.

[L43] 

[L44] ![mceclip11.png](https://support.optisigns.com/hc/article_attachments/1500001932301)

[L45] 

[L46] For the Department A Playlist folder in Playlist: You can edit the "Change Permission" in the folder. You can set it to **DepartmentA User**.

[L47] 

[L48] ![mceclip14.png](https://support.optisigns.com/hc/article_attachments/1500001932661)

[L49] 

[L50] For the Department B Playlist folder in Playlist: You can edit the "Change Permission" in the folder. You can set it to **DepartmentB User**.

[L51] 

[L52] ![mceclip19.png](https://support.optisigns.com/hc/article_attachments/1500001932781)

[L53] 

[L54] Now we have:┬Ā  

[L55] Admin can see:

[L56] 

[L57] * Administration folder in the Screen.

[L58] * Department A and Department B folder in the Files/Assets

[L59] * Department A and Department B folder in the Playlist

[L60] 

[L61] DepartmentA User can see the

[L62] 

[L63] * Department A folder in the Files/Assets

[L64] * Department A Playlist Folder in the Playlist

[L65] 

[L66] DepartmentB User can see the

[L67] 

[L68] * Department B folder in the Files/Assets

[L69] * Department B Playlist Folder in the Playlist
