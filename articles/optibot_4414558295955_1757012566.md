[L1] # Tutorial: Create, Update, Add, Remove items from Playlists

[L2] 

[L3] In this tutorial, we will walk through how to create a playlist and manage the assets assigned to the playlist through the API. Generally, it involves 4 steps. ┬ĀFirstly, you will need to create a playlist. Secondly, you will need to add content to the playlist. Then you can update the playlist items to change the setting of the playlist items, e.g. change the play duration. Lastly, you can remove the contents from the playlist.

[L4] 

[L5] #### **1. Create a playlist**

[L6] 

[L7] To create a playlist, you will need to use the savePlaylist mutation. And supply the playlist name as a arguments in the payload. If it runs successfully, it will return the requested data from the server, and an id will be assigned to the playlist. This id will be needed in the next step when assigning contents to the playlist.

[L8] 

[L9] ```

[L10] mutation {  

[L11]   savePlaylist(payload:{name:"demo\_playlist"}){  

[L12]           _id,  

[L13]           name,  

[L14]           path,  

[L15]           assets{  

[L16]             _id,  

[L17]           	fileType,  

[L18]             filename,  

[L19]           	duration,  

[L20]           	appType  

[L21]           },  

[L22]           teamId,  

[L23]           tags  

[L24]   }  

[L25] }

[L26] ```

[L30] #### **2. Assign contents to the playlist**

[L31] 

[L32] To assign the contents to the playlist, you can use the addPlaylistItems mutation. You will need to supply the playlist id retrieved from the above step, and also provide the asset id in the payload.

[L33] 

[L34] ```

[L35] mutation {  

[L36]   addPlaylistItems(_id:"4Xt7P6eYJGc4bXjR8",payload:{  

[L37]     ids: ["uRQynMhDsJ6QY35Wf",  

[L38]  "6744ff5add229e00123cf5e6"],  

[L39]     pos: 1  

[L40]   }){  

[L41]           	_id,  

[L42]           	fileType,  

[L43]           	filename,  

[L44]           	duration,  

[L45]           	appType  

[L46]   }  

[L47] }

[L48] ```

[L49] 

[L50] ![](https://support.optisigns.com/hc/article_attachments/36563843298579)

[L51] 

[L52] #### **3. Change the duration of the content in the playlist**

[L53] 

[L54] To change the duration of the assets in the playlist, you can use the updatePlaylistItems mutation. You will need to supply the playlist id retrieved from the above step, and also provide the duration and position of the asset in the playlist that you want to apply the change to in the payload.

[L55] 

[L56] ```

[L57] mutation {  

[L58]   updatePlaylistItems(_id:"4Xt7P6eYJGc4bXjR8",payload:{  

[L59]     items:  [{item: {duration:15}, pos: [0,1]}]  

[L60]   }){  

[L61]             _id,  

[L62]             fileType,  

[L63]             filename,  

[L64]             duration,  

[L65]             appType  

[L66]   }  

[L67] }

[L68] ```

[L69] 

[L70] ![](https://support.optisigns.com/hc/article_attachments/36563843301523)

[L71] 

[L72] Now, you can see the demo\_playlist is created in the account with the selected assets. And the duration of the items are set to 15 seconds.

[L76] #### **4. Remove the content from the playlist**

[L77] 

[L78] To remove the contents from the playlist, you can use the removePlaylistItems mutation. You will need to supply the playlist id retrieved from the above step, and also provide the position of the asset in the playlist that you want to remove in the payload.

[L79] 

[L80] ```

[L81] mutation {  

[L82]   removePlaylistItems(_id:"d87B9ARKPyH8YYBbs",payload:{pos: [0,1]}){  

[L83]             _id,  

[L84]             fileType,  

[L85]             filename,  

[L86]             duration,  

[L87]             appType  

[L88]   }  

[L89] }

[L90] ```

[L91] 

[L92] ![](https://support.optisigns.com/hc/article_attachments/36563851134995)

[L93] 

[L94] Now you can see, the first 2 assets in the playlist are removed from the playlist.
