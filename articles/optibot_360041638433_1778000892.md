[L1] # How to use the CountDown App

[L2] 

[L3] ### In this article, we'll explain how to use the CountDown app on OptiSigns.

[L4] 

[L5] * [What You'll Need](#WhatYouNeed)

[L9] 

[L10] The CountDown (or Count Down) app is a simple way to display a customizable count down clock for any occasion.┬Ā

[L11] 

[L12] ![OptiSigns count down timer example](https://support.optisigns.com/hc/article_attachments/45147769373075)

[L13] 

[L14] The clock can display any title you like, and when it goes off, can display any message you like. Here, we'll show how to create one. Let's get started.

[L15] 

[L16] ---

[L17] 

[L18] ## What You'll Need

[L19] 

[L20] * An [**OptiSigns-capable device**](https://support.optisigns.com/hc/en-us/articles/360021855653-What-hardware-and-devices-are-supported)

[L21] * An OptiSigns [**Standard plan or higher**](https://www.optisigns.com/pricing)

[L22] * A screen, [**set up and paired**](https://support.optisigns.com/hc/en-us/articles/18823504383891-OptiSigns-Getting-Started-Guide)

[L25] 

[L26] ## Creating a CountDown Asset

[L27] 

[L28] From the OptiSigns portal, go to **Files/Assets** **ŌåÆ Apps:**

[L29] 

[L30] ![optisigns files and assets](https://support.optisigns.com/hc/article_attachments/45147769375379)

[L31] 

[L32] On the Apps menu, select **CountDown:**

[L33] 

[L34] ![Countdown app location in optisigns](https://support.optisigns.com/hc/article_attachments/45147743659795)

[L35] 

[L36] The below screen should open up:

[L37] 

[L38] ![countdown app appearance in optisigns](https://support.optisigns.com/hc/article_attachments/45147743661203)

[L39] 

[L40] These are the options, and what they do. Any edits made to these options should be automatically reflected in the **Preview** pane on the right. The **Preview** can be displayed in either **Landscape** or **Portrait** format.

[L41] 

[L42] * **Name:** The name of the CountDown asset. This is for saving the asset in OptiSigns, and the text entered here will not appear on the asset itself.

[L43] * **Text:** The text to be displayed on the CountDown asset while it counts down. By default, this says "OptiSigns Timer," and it is in this area this message will be displayed.

[L44] * **End Time:** When selected, will bring up a calendar:

[L45] 

[L46] ![calendar end time](https://support.optisigns.com/hc/article_attachments/45147743663379)

[L47] 

[L48] This allows you to choose the precise date and time for the countdown to be completed.

[L49] * **Message:** The message to be displayed at the end of the countdown.

[L50] * **Timer Style:** Lets you choose the type of Timer to display. Choose between **Flip**, **Basic**, **Ring**, and **Slide:**

[L51] 

[L52]   

[L53] ![timer styles count down app](https://support.optisigns.com/hc/article_attachments/45147743666323)  

[L54] ┬Ā

[L55] 

[L56] Specific differences between these can be seen in the Preview panel.

[L57] * **Days Only:** Check this box to only show how many days are left on your countdown timer.

[L58] * **Theme:** Lets you change between a number of basic themes. Scroll down for more. Their specifics can be seen in the Preview panel.

[L59]

[L61] 

[L62]   

[L63] ![theme settings count down app](https://support.optisigns.com/hc/article_attachments/45147743672851)

[L65] * **Primary Color:** Determines the background color. Can be chosen with Hex Code or via color picker:

[L66] 

[L67]   

[L68] ![colour picker countdown app](https://support.optisigns.com/hc/article_attachments/45147769390611)

[L69] * **Secondary Color:** Determines the text color. Can be chosen with Hex Code or via color picker.

[L70] * **Font Size:** Choose between Default Font Size, or Custom. When Custom is selected, will provide a new option: **Custom Font Size**.

[L71] 

[L72]   

[L73] **![font size countdown app](https://support.optisigns.com/hc/article_attachments/45147743678099)**

[L74] 

[L75] 	+ **Custom Font Size:** Choose your font size.

[L76] * **Transparency:** Slider which allows the transparency of the image to be changed.

[L82] ### **Advanced Settings**

[L83] 

[L84] Expandable panel. Click this for even more options:

[L85] 

[L86]   

[L87] ![advanced settings countdown app](https://support.optisigns.com/hc/article_attachments/45147769396883)

[L88] 

[L89] * **Recurring:** When checked, enables recurring events to be repeated at selected intervals. Opens up more options:

[L90] 

[L91]   

[L92] ![recurring box countdown app](https://support.optisigns.com/hc/article_attachments/45147769398163)

[L93] 

[L94] 	+ **Repeat Every:** Lets you choose how often the countdown repeats. You can adjust this number, and choose from **Day, Week, Month,** or **Year**. When **Week, Month,** or **Year** is chosen, it will show what days it will **Repeat On**.

[L95] 	

[L96] 	

[L97] 	  

[L98] 	  

[L99] 	**Week:**  

[L100] 	![repeat on weekly](https://support.optisigns.com/hc/article_attachments/45147743684371)  

[L101] 	**Month:**  

[L102] 	![repeat on monthly](https://support.optisigns.com/hc/article_attachments/45147769400211)  

[L103] 	**Year:**  

[L104] 	**![repeat on yearly](https://support.optisigns.com/hc/article_attachments/45147743686419)**

[L105] 	+ **Ends:** Lets you choose when a recurring countdown timer will end. This can be **Never**, **On**, or **After**.

[L106] 		- **Never:** The countdown timer will repeat as per above instructions and will not stop recurring until asset is changed.

[L107] 		- **On:** Allows you to set a specific date at which time the countdown timer will stop recurring.

[L108] 		- **After:** Allows you to set a specific number of recurrences after which the timer will stop recurring.

[L117] * **Show Date:** When checked, will show the date on the countdown timer. Also provides **Date Format** option.

[L118] 

[L119]

[L122] 	+ **Date Format:** Allows you to choose a format for the displayed date.

[L123] * **Background Image:** Allows you to choose between a **Default** Background Image and a **Custom** one. Selecting **Custom** will enable the **Choose Photo** option.  

[L124] ![choose photo optisigns countdown app](https://support.optisigns.com/hc/article_attachments/45147743690643)

[L125] 	+ **Choose Photo:** Lets you use an existing OptiSigns image asset as a Background Image for your countdown timer. This will override any other Background Image options, including Background Color, etc.

[L126] * **Language:** Choose the language to display clock details such as **Hour**, **Minute**, and **Date** from a variety of language options. [L128] After you've configured the CountDown app to your liking, hit **Save**. Your CountDown app will appear as an Asset in OptiSigns, and it can now be directly assigned to a screen or used in a Playlist. This asset can then be edited at any time from the Files/Assets tab.
