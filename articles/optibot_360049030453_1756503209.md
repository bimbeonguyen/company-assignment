[L1] # How to use the Motivational Quote App

[L2] 

[L3] This article will show you how to set up and display the Motivational Quote app on your big screens TVs.

[L21] 

[L22] The following window will pop-up and you will need to enter the details and quotes you'd like to display:

[L23] 

[L24] ¬†![mceclip5.png](https://support.optisigns.com/hc/article_attachments/360072447053)

[L25] 

[L26] * Name: Name of your Motivational Quote app that you wish to see it labeled as within the OptiSigns platform. This name is internal to you only.

[L27] 

[L28] * Background Type:¬† The drop-down will show you either Playlist or Single Color

[L29] 	+ Playlist - This allows you to select an playlist of images that you want to appear as backgrounds to the quotes provided. Background Opacity and Text Color also become enabled.

[L30] 	+ Single Color - This allows you to select a single color background based on the default themes provided through OptiSigns.

[L31] * Background Opacity: This appears only when Playlist is selected and allows user to customize and optimize the opacity of the background so that the text still stands out effectively.

[L32] * Text Color:¬† This appears only when a Playlist is selected and allows user to enter the hex-code for the text color

[L33] 

[L34] * Quote:¬† Free-form text field to allow you to enter the quote and the author of the quote. Additional quotes can be added using the "+ New Quote" button

[L35] * Duration (sec): Enter the duration you'd like to have for each quote to be displayed.

[L36] * Show Date/Time: Enable a center justified date and time to be displayed above the quotes.

[L37] * Shuffle Play: play quotes in random orders.

[L38] 

[L39] ## **That's all! Congratulations!**

[L40] 

[L41] You have created an instance of the¬† Motivational Quote app.

[L42] 

[L43] You can assign it to your screen by going to Screens, click Edit screens and assign the wall directly to a screen or add it to a playlist.
