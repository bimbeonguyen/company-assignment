[L3] |  |

[L4] | --- |

[L6] 

[L7] Content Rotation App will allow you to mix content between 2 or more playlists and ensure a distribution of screen time for each playlists.

[L8] 

[L9] A real world scenario would be: there are 2 Departments each maintain a playlist: Department A and Department B.  

[L10] As Admin, you want both to have 50% of screen time each (i.e. ensure 1 department do not make long content, video and consume all the screen time).

[L11]

[L14] ## **Let's jump in and get started:**

[L15] 

[L16] First, you will need to have two or more playlists. For more information on how to do that, click [here](https://support.optisigns.com/hc/en-us/articles/28295104605843).

[L17] 

[L18] Go to Files/Assets, Click on "App".

[L19] 

[L20] ![mceclip0.png](https://support.optisigns.com/hc/article_attachments/360100601834)

[L21] 

[L22] Click Content Rotation app:

[L23] 

[L24] ![mceclip1.png](https://support.optisigns.com/hc/article_attachments/1500001706142)

[L25] 

[L26] Click Add Playlist to add the playlist you want to rotate, in this example we put 2 playlists: Department A, Department B. You can add as many playlist as you want.

[L27] 

[L28] ![mceclip0.png](https://support.optisigns.com/hc/article_attachments/1500001572662)

[L29] 

[L30] * Name: This is the name to organize assets, it will not be shown on the screen.

[L31] * Add Playlist:

[L32] 

[L33] ![mceclip2.png](https://support.optisigns.com/hc/article_attachments/360102767433)

[L34] 

[L35] * + Playlist: Select your playlist.

[L36] 	+ Rotate Every\*: 1 means the system will play 1 items on this playlist then move on to next playlist

[L37] 	+ Each Item Duration\*: Set duration for each items, this is required in case one playlist has a very long video for example, this will ensure screen time is still allocated fairly between playlists.

[L38] 

[L39] ![mceclip1.png](https://support.optisigns.com/hc/article_attachments/1500001572702)

[L40] 

[L41] * Enforce each item duration limit: Force each asset's playing time is same as Each Item Duration time. When this is checked, you will need to select how you want to handle when an item's duration is less than "Each item duration":

[L42] 	+ Move to next: If an item's duration is less than "Each item duration", it will move to next

[L43] 	+ Repeat: If an item's duration is less than "Each item duration", it will repeat the same asset until the "Each item duration" is over.

[L44] 

[L45] Click Save.  

[L46] After Saving, you can Preview the Content Rotation to see how the system mix the playlist.

[L47] 

[L48] ┬á

[L49] 

[L50] ## **That's all! Congratulation!**

[L51] 

[L52] You have added a Content Rotation app.

[L53] 

[L54] You can assign the newly created Content Rotation asset to your screen by going to Screens, click Edit screens and assign the asset to screens that you want.
