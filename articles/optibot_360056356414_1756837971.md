[L1] # How to use Scrolling Website App

[L2] 

[L3] Sometimes you have requirement to show a certain website and have it scroll or zoom in a certain way.  

[L4] Scrolling Website App is the solution for that, it will allow you to:

[L5] 

[L6] * Use any Website URL

[L7] * Set the app to scroll down & up the web page, or just scroll to a certain location

[L8] * Set scrolling speed

[L9] * Set the app to zoom in/out

[L10] 

[L11] ## Let's jump in and get started:

[L12] 

[L13] ┬ĀGo to File/Asset -> App and select Scrolling Website App:

[L14] 

[L15] ![mceclip0.png](https://support.optisigns.com/hc/article_attachments/360095655314)

[L16] 

[L17] ┬Ā

[L18] 

[L19] Set up your Scrolling Website App:

[L20] 

[L21] ![mceclip1.png](https://support.optisigns.com/hc/article_attachments/360097955053)

[L22] 

[L23] * Name: name of your Scrolling Website App, this is only for use in File/Asset list, it will not be displayed on your screen.

[L24] * URL: the URL link to the page you want to display.

[L25] * Request Desktop Site: if you are using Amazon Fire Stick or Android as your player, if this box is checked, the app will try to request desktop version of the web page.

[L26] * Scroll Type: You can select

[L27] 	+ - * Down & Up: the app will scroll the web page down, and then scroll up.

[L28] 			* Scroll to: the app will jump to a certain location on the web page

[L29] * Speed: how fast you want to scroll, you can also customize the speed.

[L30] * Page Height: enter the number in Pixel. This is needed because the app does not know how long a web page is, also some web page like social media has infinite scrolling.

[L31] 

[L32] Click Save to save your asset.

[L33] 

[L34] You can preview how it look by clicking the "Preview button".

[L35] 

[L36] ┬Ā

[L37] 

[L38] ## That's all!

[L39] 

[L40] Congratulation! You have created your Scrolling Website App.

[L41] 

[L42] You can change the wall any time by click on it in the Files/Assets tab.

[L43] 

[L44] ┬Ā

[L45] You can assign the newly created app to your screen by going to Screens, click Edit screens and assign the wall to screens that you want.

[L46] 

[L47] You can put the walls in a Playlist, Schedule too.
