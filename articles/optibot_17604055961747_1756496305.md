[L19] ## What is a Content Library Kiosk?

[L20] 

[L21] Think of it as organizing computer files with folders:

[L22] 

[L23] * You create a Root folder on the Files/Assets page, which serves as the central directory for all your content.

[L24] * Within the root folder, you create Subfolders that act as categories or themes, providing an organized structure.

[L25] * Your digital assets are placed in these subfolders, typically grouping together similar items or materials related to the folder's theme. This organization makes it easy to find and access specific content, creating an organized and user-friendly experience for your audience.

[L26] 

[L27] Now that you have everything digitally organized, you're ready to transform it into an interactive kiosk for your customers to engage with.

[L41]  **Creating a Content Library Kiosk**

[L42] 

[L43] Before you can create the Content Library Kiosk app, you must first establish a content library. This library should be organized or prepared on our Files/Asset page, as specifying the Root folder is a required step.

[L44] 

[L45] Once the library is ready, you can begin to create your content library kiosk app. Please follow steps below:

[L46] 

[L47] Go to Engage, then click **New App.**

[L48] 

[L49] ![](https://support.optisigns.com/hc/article_attachments/23035917641363)

[L50] 

[L51] Select **Touch Screen Kiosks**. Then, go to the next step to **Select App.**  

[L52] ![](https://support.optisigns.com/hc/article_attachments/23696559854227)

[L53] 

[L54] On the Select App page, click on **Content Library Kiosk app** then continue to **Build**.

[L55] 

[L56] ![](https://support.optisigns.com/hc/article_attachments/23696533418515)

[L57] 

[L58] On the **Build** page, you will have several key features that you can customize and configure. Let's dive into each configuration detail:

[L59] 

[L60] ![](https://support.optisigns.com/hc/article_attachments/23696559863443)

[L61] 

[L62] * **Name**: A unique identifier for the asset, not visible on the kiosk

[L63] * **Header**: The title displayed on the content library kiosk.

[L64] * **Custom Logo**: A personalized logo that can be added to the interface.

[L65] * **Root Folder**: The main directory where content is organized. Required field.

[L66] * **Featured Folder**: A designated directory showcasing highlighted or prominently displayed content. You can only choose featured folders that are within your root folder. In Carousel Explorer View, the featured folder will be displayed at the top of the kiosk page.

[L67] * **Display Mode**: Choose a layout for showcasing content.

[L68] 	+ **Carousel Explorer View** - Folders within the root folder are presented in a rotating carousel format, while individual contents within the root folder will display below it as tiles.

[L69] 	+ **Content Explorer View** -Folders within the root folder are presented at the top as long tiles, with individual contents within the root folder displayed below as tiles.

[L70] 	+ **Tile View** - All contents, regardless of whether they are in folders within the root folder, will be displayed as individual items and in tiles.

[L71] * **Portrait Mode**: This setting will turn all your content orientation to a vertical display.

[L72] * **Sort By**: Criterion on how your content will be displayed on the kiosk based on the assets created within the folders.

[L73] * **When No Network**: Options for behavior when the kiosk is offline.

[L74] 	+ **Do nothing** - Leave as-is

[L75] 	+ **Disable Web Assets** - Any assets requiring the web will still be visible but disabled until the network is restored.

[L76] 	+ **Hide Web Assets** - Any assets requiring the web will hidden until the network is restored.

[L77] * **Close Icon Location** - Location of where you would like the Close (X) icon to display on your kiosk screen; choose 'None' if you do not want the close icon to display.

[L78] * **Theme:** The visual style applied to the kiosk interface.

[L79] * **Font Size:** Specify the size of the text within the interface; set to Default

[L80] * **Font┬Ā Family**: The chosen typeface for text elements.

[L81] 

[L82] **Advanced Settings**:

[L83] 

[L84] ![](https://support.optisigns.com/hc/article_attachments/23696559863955)

[L85] 

[L86] * **Timeout Asset Viewing**: Set duration in seconds before automatic closure of displayed assets

[L87] * **Timeout Browsing**: Set a time limit in seconds for user inactivity before returning to the main screen.

[L88] * **Background Image**: Customize the image for the kiosk background.

[L89] 

[L90] Once you have set your configuration settings, click **Review** to continue to the next page.┬Ā

[L91] 

[L92] * On the Review page, you can review all of your configuration settings and preview the screen before committing your changes. You can also preview in full screen by clicking on 'Preview Full Screen' at the bottom.

[L93] 

[L94] ![](https://support.optisigns.com/hc/article_attachments/23696476261267)

[L95] 

[L96] Click **Assign** to continue to the next page.

[L97] 

[L98] * On this page, you can assign your content library kiosk to a Screen or Screens with a specific tag.

[L99] * You can also choose to Assign Later.

[L100] 

[L101] ![](https://support.optisigns.com/hc/article_attachments/29266141478291)

[L102] 

[L103] When you have signed your Screens, click **Complete**.

[L104] 

[L105] ![](https://support.optisigns.com/hc/article_attachments/23696476280083)
