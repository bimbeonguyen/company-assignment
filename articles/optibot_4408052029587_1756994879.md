[L1] # How to Use Weather Apps Based on Device Location

[L2] 

[L3] If you have many screens in different locations, and you want to display local weather, you can just make 1 weather app, and set it to use device's location instead of having to make many weather apps.

[L4] 

[L5]  

[L6] 

[L7] First set your device's location.

[L8] 

[L9] In Edit screen, click Advanced, and add location.

[L10] 

[L11] ![mceclip1.png](https://support.optisigns.com/hc/article_attachments/4408051931283)

[L12] 

[L13] You can also edit location of multiple screens at the same time by selecting them and click Edit

[L14] 

[L15] ![mceclip2.png](https://support.optisigns.com/hc/article_attachments/4408057893011)

[L16] 

[L17]  

[L18] 

[L19] Next, in your weather app select the option to "Use Device Setting's Location".

[L20] 

[L21] You still need to set the default location, this location will be used in case device does not have location set up.

[L22] 

[L23] ![](https://support.optisigns.com/hc/article_attachments/36361308056211)

[L24] 

[L25] #### For more information on weather apps, see our article on [how to use weather apps in OptiSigns](https://support.optisigns.com/hc/en-us/articles/360017964153-How-to-Use-Weather-Apps-in-OptiSigns).

[L26] 

[L27] #### **That's all!**

[L28] 

[L29] Now you can use this Weather app in your playlist, schedule, etc. and when the device plays, it will use the location's set in its settings to get Weather information.
