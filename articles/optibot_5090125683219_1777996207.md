[L1] # BrightSign Players

[L2] 

[L3] BrightSign is the leader in commercial grade digital signage players, which is known for its reliability. OptiSigns provides full support on BrightSign player, however we do recommend using the HD, XD, and XT product lines for better performance.

[L4] 

[L5] It is very easy to setup OptiSigns on BrightSign players. There are 2 ways you can set it up. You can set it up in standalone mode, or you can set it up on BSN.cloud.

[L6] 

[L7] ## Option 1: Setup with BSN.cloud

[L8] 

[L9] **1) Download BrightAuthor**

[L10] 

[L11] The latest BrightAuthor can be downloaded [here](https://www.brightsign.biz/digital-signage-products/software/brightauthor).

[L12] 

[L13] **2) Logon to your BSN.cloud account and select device setup**

[L14] 

[L15] ![mceclip3.png](https://support.optisigns.com/hc/article_attachments/5144414949011)

[L16] 

[L17] **3) Complete the device setup**

[L18] 

[L19] Fill in the value in the package and device name.

[L20] 

[L21] Select "Partner Application" under Device Type. Click "Select a Partner Application" and find "OptiSigns".

[L22] 

[L23] Then click "Save Setup File" to save the generated setup files.┬Ā

[L24] 

[L25] ![mceclip1.png](https://support.optisigns.com/hc/article_attachments/5200010385299)

[L26] 

[L27] ![mceclip2.png](https://support.optisigns.com/hc/article_attachments/5200003714195)

[L28] 

[L29] **4) Copy the setup files to your storage device.**

[L30] 

[L31] Place the generated setup files in the root directory of your SD card.

[L32] 

[L33] ![mceclip2.png](https://support.optisigns.com/hc/article_attachments/5144369828755)

[L34] 

[L35] **5) Insert the SD card to the player and let it finish the setup**

[L36] 

[L37] The BrightSign player will read the setup files and complete the setup automatically. It will take a few minutes and may reboot a few times.

[L38] 

[L39] Once the setup process is complete, you will see OptiSigns player running on your screen with the pairing code showing up on the screen.

[L40] 

[L41] Please make sure your BrightSign player is connected to the internet.

[L42] 

[L43] ![mceclip1.png](https://support.optisigns.com/hc/article_attachments/5089996700435)┬Ā**6) Pair the screen**Now you are ready to go to our portal at:┬Ā<https://app.optisigns.com/>

[L44] Pair the screen using your screen's code and start assigning content to it.For detailed steps click┬Ā[here](https://support.optisigns.com/hc/en-us/articles/360016374813).

[L48] ## Option 2: Standalone Mode

[L49] 

[L50] **1) Download the autorun file.**

[L51] 

[L52] The latest autorun.zip file can be downloaded [here](https://links.optisigns.com/brightsign).┬Ā

[L53] 

[L54] **2) Copy the autorun file to your storage device.**

[L55] 

[L56] The autorun.zip file should be placed in the root of your SD card. The SD card needs to be formatted to **FAT32**. NTFS format does not work for this setup

[L57] 

[L58] ![mceclip2.png](https://support.optisigns.com/hc/article_attachments/5090013459347)

[L59] 

[L60] **3) Insert the SD card to the player and let it finish the setup**

[L61] 

[L62] The BrightSign player will read the data from the autorun file and complete the setup automatically. It will take a few minutes and may reboot a few times.

[L63] 

[L64] Once the setup process is complete, you will see OptiSigns player running on your screen with the pairing code showing up on the screen.

[L65] 

[L66] Please make sure your BrightSign player is connected to the internet.

[L67] 

[L68] ![mceclip1.png](https://support.optisigns.com/hc/article_attachments/5089996700435)┬Ā**4) Pair the screen**Now you are ready to go to our portal at:┬Ā<https://app.optisigns.com/>

[L76] ## Updating BrightSign Software

[L77] 

[L78] If you have further issues running OptiSigns, it's possible you have an outdated version of the BrightSign software. We recommend updating it.

[L79] 

[L80] The latest version of BrightSign software can be found [here](https://www.brightsign.biz/resources/software-downloads/).

[L81] 

[L82] ## How to Update the BrightSign Registry to Use Newer Versions of Chromium

[L83] 

[L84] A common issue we find is that BrightSign may need a registry update in order to display newer versions of Chromium. This is a particular issue with displaying PowerBI.

[L85] 

[L86] To do this, open up BrightAuthor:connected and navigate to **Admin:**

[L87] 

[L88] ![](https://support.optisigns.com/hc/article_attachments/36888690739731)

[L89] 

[L90] Now you'll need to obtain the IP of your BrightSign player. Input the IP address into your browser (your device must be on the same network as the BrightSign player). If you've done things correctly, you will enter the player config menu.

[L91] 

[L92] Once there, navigate to **Registry**:

[L93] 

[L94] ![](https://support.optisigns.com/hc/article_attachments/36888690740627)

[L95] 

[L96] From this

[L97] 

[L98] ![](https://support.optisigns.com/hc/article_attachments/36888690742547)

[L99] 

[L100] Next, input the following command:

[L101] 

[L102] ```

[L103] registry write html widget_type chromium110

[L104] ```

[L105] ![](https://support.optisigns.com/hc/article_attachments/36888685278483)

[L106] 

[L107] When you hit submit, you'll see this:

[L108] 

[L109] ![](https://support.optisigns.com/hc/article_attachments/36888690746387)

[L110] 

[L111] This will allow your BrightSign player to use the latest versions of Chromium.
