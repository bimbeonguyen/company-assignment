[L1] # Add 1 or multiple assets to many playlists at the same time

[L2] 

[L3] Sometimes you just create new assets and want to add them to several playlists, you can do it quickly:

[L4] 

[L5] Click ... on the asset, then click Add to Playlists

[L6] 

[L7] ![mceclip0.png](https://support.optisigns.com/hc/article_attachments/4404608925843)

[L8] 

[L9]  

[L10] 

[L11] You can also select multiple assets and click Add to Playlist button.

[L12] 

[L13] ![mceclip2.png](https://support.optisigns.com/hc/article_attachments/4404616134675)

[L14] 

[L15]  

[L16] 

[L17] Then select all the playlists you want to add to and click Save.

[L18] 

[L19] ![mceclip1.png](https://support.optisigns.com/hc/article_attachments/4404616130579)

[L20] 

[L21] The assets will be added to the end of the playlists.

[L22] 

[L23] That's it, you have added assets to many playlists at the same time.
