[L1] # How to use Microsoft Excel Online with OptiSigns

[L2] 

[L3] Microsoft Excel is a very popular ways to create and share your spreadsheet. You can also use it on your Digital Signage screen.

[L4] 

[L5] There are 2 ways to use Excel with OptiSigns:

[L6] 

[L7] 1) Upload the XLSX file, and OptiSigns will convert and display on your screen (pros: can play offline, just upload files, cons: when there are changes, you need to upload the files again)

[L10] 

[L11] 2) Use Excel for Web (online) (pros: **you can make changes to the Excel file and screens will update automatically**, cons: need internet connection to display)

[L17] Upload your Excel to OneDrive and changes to your Excel can be automatically updated on your screens.

[L49] ## **2) Add Excel Online App on OptiSigns**

[L50] 

[L51] Go to our portal: <https://app.optisigns.com>

[L52] 

[L53] Click File/Assets, then click Apps, search and select Excel Online app.

[L55] 

[L56] There are two options to enter your Microsoft Excel.

[L57]

[L61] **Option 1: Copy and paste your Microsoft Excel embed code directly.**

[L109] ### Why does using Excel require giving OptiSigns Admin permission to use?

[L110] 

[L111] OptiSigns uses Microsoft APIs for integration. In order for our integrations to work, the integration has to be approved by an administrator. This is the same across all integrations using Microsoft APIs.

[L112] 

[L113] This administrator access is only needed for first time access. Once the OptiSigns app is approved for use, other users can use OptiSigns directly.
