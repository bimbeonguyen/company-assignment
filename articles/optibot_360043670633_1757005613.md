[L1] # Organizing your files, assets, playlists and schedules by using Folders

[L2] 

[L3] You can create folder to organize your files, assets, playlists and schedules.

[L4] 

[L5] If you have a lot of content, creating folders will help you organize them. You can also restrict access to them using [**Folder****Security**](https://support.optisigns.com/hc/en-us/articles/360044600474-Advanced-Security-Folder-Level-Security).

[L6] 

[L7] To create a new folder for Assets, just click on the "New Folder" button in the File/Asset tab.

[L8]

[L13] To create a new folder for Playlists, click the "New Folder" button in the Playlist tab.
