[L1] # How to run OptiSigns AI Add-on on start up in Windows

[L2] 

[L3] In Windows, you can use Windows' Task Scheduler to set up OptiSigns AI Add-on to run on start up.

[L4] 

[L5] It only take a few simple steps

[L6] 

[L7] Click **Start** -> Type "**Task Scheduler**" and open it

[L8] 

[L9] Then click **Create Basic task**

[L10] 

[L11] ![mceclip0.png](https://support.optisigns.com/hc/article_attachments/360100967213)

[L12] 

[L13] Give it a name (e.g. OptiSigns AI Add-on Autostart)

[L14] 

[L15] Then click **Next**

[L16] 

[L17] ![mceclip1.png](https://support.optisigns.com/hc/article_attachments/360098751914)

[L18] 

[L19] Next in Trigger section select "**When I Log on**"  

[L20] Please do not select other event like When computer start, it may not work.

[L21] 

[L22] ![mceclip2.png](https://support.optisigns.com/hc/article_attachments/360100967293)

[L23] 

[L24] Next in Action section select "**Start a program**"

[L25] 

[L26] ![mceclip3.png](https://support.optisigns.com/hc/article_attachments/360098752054)

[L27] 

[L28] Click Browse and browse to optisigns-ai-detection.exe that you've downloaded.

[L29] 

[L30] ![mceclip4.png](https://support.optisigns.com/hc/article_attachments/360098752114)

[L31] 

[L32] Then click **Finish**

[L33] 

[L34] ![mceclip5.png](https://support.optisigns.com/hc/article_attachments/360098752154)

[L35] 

[L36]  

[L37] 

[L38] That's all!

[L39] 

[L40] Next time your PC start up, it will start OptiSigns AI Detection app, doing it this way will also ensure that it will get started before the OptiSigns Digital Signage Player app start.
