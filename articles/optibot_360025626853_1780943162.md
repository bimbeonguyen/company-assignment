[L1] # How to start OptiSigns automatically when devices boot up

[L2] 

[L3] When you start the app for the first time.  

[L4] It will ask if you want the app to start with device start up.  

[L5] You can select Yes.

[L6] 

[L7] ![mceclip0.png](https://support.optisigns.com/hc/article_attachments/360040341113)

[L8] 

[L9] You can change this anytime by open the side menu.  

[L10] And select Autostart.

[L11] 

[L12] ![mceclip0.png](https://support.optisigns.com/hc/article_attachments/360047734253)

[L13] 

[L14] On Fire TV, you can open side menu by clicking the 3 bars button (see below).

[L15]
