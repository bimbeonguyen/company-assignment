[L1] # Assign Default Content for Schedules

[L2] 

[L3] If you create a schedule, at times when there's no content scheduled, the display will default to a black screen.  

[L4] For example the schedule below, at 7AM the screen will just display a black screen.  

[L5] Sometime, you would want to set a default content so that when nothing is scheduled the default content will be displayed.

[L6] 

[L7] There are 2 ways you can setup default content:

[L8] 

[L9] 1. At the schedule level (apply to a schedule only). To do so, click on the Schedule you want to edit, then click Option.

[L10] 

[L11] ![](https://support.optisigns.com/hc/article_attachments/51444803740435)

[L12] 

[L13] Then select if you want to use:

[L14] 

[L15] ![](https://support.optisigns.com/hc/article_attachments/51444826883859)

[L16] 

[L17] * **None:**just a black screen when nothing is scheduled

[L18] * **Use Account Default:** using what set at account level (see #2 below for how).

[L19] * **Asset:** select and asset to be displayed when nothing is scheduled

[L20] * **Playlist:** select a playlist to be played when nothing is scheduled

[L21] 

[L22] 2. Or at Account level so it will apply to all **newly created schedule.** (In case you both at account level and at schedule level, the schedule level setting will take priority for that schedule).

[L23] 

[L24] ![](https://support.optisigns.com/hc/article_attachments/51444803744659)

[L25] 

[L26] Then select if you want to use:

[L27] 

[L28] ![](https://support.optisigns.com/hc/article_attachments/51444826887315)

[L29] 

[L30] * **None:**just a black screen when nothing is scheduled

[L31] * **Asset:** select and asset to be displayed when nothing is scheduled

[L32] * **Playlist:** select a playlist to be played when nothing is scheduled

[L33] 

[L34] ### Summary:

[L35] 

[L36] Sometime, you would want to set a default content so that when nothing is scheduled the default content will be displayed. There are 2 ways to set default content, one is at each schedule level, and one is at whole account level and will be applied to all newly created schedule.
