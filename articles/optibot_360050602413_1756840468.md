[L1] # How to use Overlay layers with OptiSigns

[L2] 

[L3] With OptiSigns SplitScreen, you can overlay items on top of each other.  

[L4] In example below we overlay OptiSigns Logo and a QR Code on top of the screen.  

[L5] As the content play, the logo keep staying on top.  

[L6] This is particular useful if you want to overlay logo, QR Codes, but you can overlay any content.

[L7] 

[L8] ![mceclip7.png](https://support.optisigns.com/hc/article_attachments/360076448373)

[L9] 

[L10] To create overlay layer, use the Split Screen app. If you haven't use the Split Screen app before, you can read the [how to here.](https://support.optisigns.com/hc/en-us/articles/360026559573)

[L11] 

[L12] In SplitScreen Editor, you just need to add 1 more zone by clicking the + button.  

[L13] The move, resize the zone to the area you want.  

[L14] For example below, we made a small Orange zone for our overlay logo.  

[L15] Then set that zone to display the logo.

[L16] 

[L17] ![mceclip2.png](https://support.optisigns.com/hc/article_attachments/360076448133)

[L18] 

[L19] You can bring the zone layer Forward, Backward, etc to set overlay layers by using these buttons.

[L20] 

[L21] ![mceclip6.png](https://support.optisigns.com/hc/article_attachments/360075260734)

[L22] 

[L23] Tips: use a transparent PNG image if you want to have transparent background.  

[L24] Note: Transparent PNG background currently only work for Windows, Linux, Mac, Raspberry Pi. On Android & FireTV it will show as black background.
