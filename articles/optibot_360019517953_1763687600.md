[L38] ## Setting Up the Google Calendar App

[L39] 

[L40] Find and select the **Google Calendar** app within the apps list:

[L41] 

[L42] ![where to find apps tab](https://support.optisigns.com/hc/article_attachments/46695218707091)![select google calendar app](https://support.optisigns.com/hc/article_attachments/46695218707347)

[L43] 

[L44] The following window pop-up should show.  

[L45] Sign-in and authenticate the account you wish to use for Calendar sharing.

[L46] 

[L47] ![](https://support.optisigns.com/hc/article_attachments/46695218708115)¬†Once authenticated, hit **Next.**

[L48] 

[L49] Now, input the Calendar details:

[L50] 

[L51] ![](https://support.optisigns.com/hc/article_attachments/46695246726803)* **Name:** Name of your calendar that you wish to see it labeled as within the OptiSigns platform. This name is internal to you only.

[L52] * **Calendars:**¬† The dropdown will show all the calendars that are within the account. These can be calendars you created or are subscribed to.

[L53] * **Title:** Title of the calendar. This will be displayed on your screen.

[L54] * **Display Mode:** Select the calendar view

[L55] 	+ **Schedule** - Linear listing of every event in succession by time and date.

[L56] 		- Day: Get today's events.

[L57] 		- Week: Get current week's events.

[L58] 		- Month: Get current month's events.

[L59] 		- 7 Days: Get 7 upcoming days (including today's) events.

[L60] 		- 30 days: Get 30 upcoming days (including today's) events.

[L61] 	+ **Day** - Listing the current day's event activity.

[L62] 	+ **Week** - Listing the current week's event activity

[L63] 	+ **Month** - Listing the current month's event activity

[L64] * **Only Show Ongoing and Upcoming Events:** When checked, will remove any past events from the calendar display.

[L65] 

[L66] ### **Theme Settings**

[L67] 

[L68] ![](https://support.optisigns.com/hc/article_attachments/46695246728467)* Hide Background Image:Show in what circumstances to hide the background image. Hide only in **Portrait**, only in **Landscape**, or in **All** situations.

[L69] * Background Image: Toggle between the Default background image, or choose a Custom one from your Assets. Selecting **Custom** will add the following option:

[L70] 

[L71] ![](https://support.optisigns.com/hc/article_attachments/46695218710291)* Choose Photo: Choose the photo to use as the Background Image

[L72] * Theme:Change between **Light Mode** or **Dark Mode**, or choose a Custom theme. If **Custom** is selected, the following option will appear:

[L73] 

[L74] ![](https://support.optisigns.com/hc/article_attachments/46695246730899)* **Background Color:** Choose the background color for the Google Calendar theme.

[L75] * **Text Color:** Text color of the display

[L76] * **Use Your Calendar Event Color:** Pull the Calendar Event color from your Google Calendar.

[L77] * **Font Size:** Font size of the display. If **Custom** is selected, you can change the size of the font. Default Font Size is 20.

[L78] 

[L79] ### **Advanced Options**

[L80] 

[L81] ![](https://support.optisigns.com/hc/article_attachments/46695218712723)* **Time Range:** Change the range of times displayed. Choose between **Full day, 8 AM - 5 PM, 9 AM - 5 PM,** or set a **Custom** time range.

[L82] * **Slot Spacing:** Change the spacing between slots in the Google Calendar.

[L83] * **Show Location:** Show the location of your calendar's event.

[L84] * **Auto Scroll:** By default, the calendar will focus the display of events on current local time or your set time zone time. This can be less helpful if you anticipate viewers to be able to see all event activities of your calendar.¬†  

[L85] Toggle the auto scroll pans the schedule so that all times are viewable.

[L86] * **Time Format:** Change between a 24-hour time format and 12-hour time format.

[L87] 

[L88] ![](https://support.optisigns.com/hc/article_attachments/46695218712979)* Timezone: select to use Device's time zone or your specified time zone.

[L89] * Language: Change the display language for your calendar.

[L90] * Weather: Entering location information allows weather information to be overlaid on your calendar.

[L91] * Scale: Select if you would like to see your temperature in Fahrenheit or Celsius.

[L92]
