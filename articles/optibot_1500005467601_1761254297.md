[L1] # Clean install of OptiSigns Player on Windows

[L2] 

[L3] Performing a clean install of OptiSigns on Windows could help to resolve issues you experiencing.

[L4] 

[L5] |  |

[L6] | --- |

[L7] | **NOTE** |

[L8] | Be sure your version of Windows is up-to-date before trying a clean install. |

[L9] 

[L10] To perform a clean install first follow these steps to completely remove OptiSigns:

[L11] 

[L12] 1) Click Start -> Type "Remove" -> Click "Add & Remove Program"

[L13] 

[L14] 2) Find OptiSigns, click Uninstall

[L15] 

[L16] ![mceclip0.png](https://support.optisigns.com/hc/article_attachments/1500007582301)

[L17] 

[L18] 3) Delete these folders if they still exist:

[L19] 

[L20] * C:\Users\[Your UserName]\AppData\Roaming\OptiSigns Digital Signage

[L21] * C:\Users\[Your UserName]\AppData\Roaming\OptiSigns

[L22] * C:\Users\[Your UserName]\AppData\Roaming\xxxxxxxxxxx.file

[L23] 

[L24] ![](https://support.optisigns.com/hc/article_attachments/19771754440723)

[L25] 

[L26]  

[L27] 

[L28] Then you can download the latest OptiSigns Windows version here:

[L29] 

[L30] [https://www.optisigns.com/download-windows](https://www.optisigns.com/download)

[L31] 

[L32] And install it again.

[L33] 

[L34]  

[L35] 

[L36] That's all you have performed a clean uninstall and reinstall of OptiSigns player.

[L37] 

[L38] If you still experiencing issues, please contact us at support@optisigns.com
