[L28] ## Basics of the Brand Kit

[L29] 

[L30] The **Brand Kit** option can be selected from the Designer Side Menu:

[L31] 

[L32] ![](https://support.optisigns.com/hc/article_attachments/42313846696339)The most critical option here is **Enforce Branding**. This is a toggle which switches between three modes:

[L33] 

[L34] * **None -** Enforce Branding is not enabled.

[L35] * **Guideline -** Enforce Branding is enabled, but the preselected fonts and colors are only used as a guideline. Users will still be able to use any font or color.

[L36] * **Enforce -** Enforce Branding is strictly enabled. Any font or color outside of those preselected will be disabled for normal users.

[L37] 

[L38] When **Enforce Branding** is enabled, new options become available in the Brand Kit side menu: **Brand Fonts** and **Brand Colors**.

[L39] 

[L40] ### Custom Fonts

[L41] 

[L42] The Brand Kit section is where you'll upload any custom fonts. Simply click **Uploaded Fonts**.

[L43] 

[L44] ![](https://support.optisigns.com/hc/article_attachments/43979008251411)The **Font Management** screen will pop up. Click **Upload Font**, then add your font.

[L45] 

[L46] ![](https://support.optisigns.com/hc/article_attachments/43978976259603)### Fonts Management

[L47] 

[L48] The **Brand Fonts** section can be used to manage accepted fonts through the Brand Kit. To add fonts from within OptiSigns, hit **Add Brand Font**.

[L49] 

[L50] ![](https://support.optisigns.com/hc/article_attachments/42313875422611)Numerous fonts can be selected from the list of **180** fonts available on Designer:

[L51] 

[L52] ![firefox_9lRMDVaico.gif](https://support.optisigns.com/hc/article_attachments/42313875422995)Once fonts are selected, theyŌĆÖll be visible under the **Brand Fonts** section.

[L53] 

[L54] ![](https://support.optisigns.com/hc/article_attachments/42313846699539)The yellow font signifies the Default font for all users. Hovering over another font lets you set it as Default or Remove it from the approved font list.

[L55] 

[L56] With Enforce Branding enabled, only fonts on this list will be available for users when they select a font from the Font picker.
