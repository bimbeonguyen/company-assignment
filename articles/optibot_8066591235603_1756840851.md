[L1] # How to use RTSP Stream on Windows or Raspberry Pi via OptiSigns 

[L2] 

[L3] #### Nowadays, RTSP Camera Streaming is popular. You can display your RTSP Stream on your Digital Screen via OptiSigns.Support_article_-2.png

[L4] 

[L5] This article will guide you through how to create and use the RTSP Stream app on Windows or Raspberry Pi. Note: For this feature, you must run OptiSigns player version 5.3.2 or newer on your devices. You can download the latest version of the app at [https://www.optisigns.com/download](http://www.optisigns.com/download)

[L6] 

[L7] If you want to use the Fire Stick or Android Stick to display the RTSP Stream app. The setup will be different. You can click [here](https://support.optisigns.com/hc/en-us/articles/8098889840531).

[L8] 

[L9] Windows and Raspberry Pi don't support the RTSP HTML5 directly. Before you create the RTSP Stream app, you will need to transcode RTSP to HTTP. Then you can implement RTSP live streaming on the Windows and Raspberry Pi devices. The easy way of achieving RTSP HTML5 playback on Windows and Raspberry Pi is to embed VLC into the HTML5 webpage and adopt it as an HTML5 RTSP player.

[L10] 

[L11] ### **Let's jump in and get started:**

[L12] 

[L13] #### **1. How to use it on Windows and Raspberry Pi devices.**

[L14] 

[L15] Before setting up, you will need to prepare:

[L16] 

[L17] 1. Need the VLC Media Player. You can download it [here](https://www.videolan.org/vlc/index.html).

[L18] 2. RTSP URL. (Here is an example: rtsp://wowzaec2demo.streamlock.net/vod/mp4:BigBuckBunny\_115k.mp4)

[L19] 

[L20] Then you can follow these steps to set it up.

[L21] 

[L22] Step 1. Launch VLC Media Player, go to the "**Media**" > **Open Network Stream**

[L23] 

[L24] ![mceclip2.png](https://support.optisigns.com/hc/article_attachments/8066835391379)

[L25] 

[L26] Step 2. Go to the "**Network**", type in the RTSP URL to the "**Network URL**". We use this as a sample example. (rtsp://wowzaec2demo.streamlock.net/vod/mp4:BigBuckBunny\_115k.mp4)

[L27] 

[L28] ![mceclip4.png](https://support.optisigns.com/hc/article_attachments/8066888364051)

[L29] 

[L30] Step 3. Click the dropdown and select "**Stream**".

[L31] 

[L32] ![mceclip6.png](https://support.optisigns.com/hc/article_attachments/8066876184211)

[L33] 

[L34] Step 4. Click "**Next**"

[L35] 

[L36] ![mceclip7.png](https://support.optisigns.com/hc/article_attachments/8066895761171)

[L37] 

[L38] Step 5. Select "**HTTP**", and Click "**Add**"

[L39] 

[L40] ![mceclip8.png](https://support.optisigns.com/hc/article_attachments/8066948003603)

[L41] 

[L42] Step 6. Type "**9999**" in the "**Port**" and "**/stream.ogv**" in "**Path**". The address of the output stream will be the IP address of "your native server/stream.ogv". Click "**Next**" to proceed.

[L43] 

[L44] ![mceclip2.png](https://support.optisigns.com/hc/article_attachments/8099130145555)

[L45] 

[L46] Step 7. Select "**Video-Theora+Vorbis┬Ā(OGG)**". Then click "**Next**"

[L47] 

[L48] ![mceclip11.png](https://support.optisigns.com/hc/article_attachments/8066996433683)

[L49] 

[L50] Step 8. Check the output media codec at "Generated stream output string", make sure **vcodec=theo** and **mux=ogg,** otherwise manually modify. Then, click "**Stream**".

[L51] 

[L52] ![mceclip12.png](https://support.optisigns.com/hc/article_attachments/8067037167379)

[L53] 

[L54] When it is successful, it will start the run. You can see the screenshot below.

[L55] 

[L56] ![mceclip1.png](https://support.optisigns.com/hc/article_attachments/8099117325203)

[L57] 

[L58] #### **2. Now it's time to create the RTSP stream app in OptiSigns Web Portal.**

[L59] 

[L60] Note: Keep VLC running for constantly transcoding, and embedding RTSP stream HTML5 webpage.

[L61] 

[L62] Go to Files/Assets, Click on "App", search for, and add the RTSP Stream app to your Account.

[L63] 

[L64] ![mceclip0.png](https://support.optisigns.com/hc/article_attachments/8066535317907)

[L65] 

[L66] Enter your RTSP Stream App information:

[L67] 

[L68] ## mceclip3.png

[L69] 

[L70] * Name: Name of your RTSP Stream app, this is the name of the app in your asset list. It will **not**┬Ābe displayed on your screens.

[L71] * URL: the URL link will be your local IP address with your setup. (http://[Your IP address]:[Port]/[path])

[L72] 

[L73] Click Save to save your asset.

[L74] 

[L75] ## That's all!

[L76] 

[L77] Congratulation! You have created your RTSP Stream App. Then you can assign it to your Windows or Raspberry Pi device.

[L78] 

[L79] You can change the wall at any time by clicking on it in the Files/Assets tab.

[L80] 

[L81] You can put the walls in a Playlist, Schedule too.
