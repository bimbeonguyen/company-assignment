[L1] # Move Screens or Devices from One Account to Another

[L2] 

[L3] ### In this article, we will walk you through the process of moving a screen or device from one OptiSigns account to another.

[L4] 

[L5] * [Step 1: Unpair the Screen from Current Account](#Step1)

[L6] 	+ [Using OptiSigns Player (With Device Access)](#Player)

[L7] 	+ [Using OptiSigns Web Portal (Remote Access)](#Portal)

[L8] * [Step 2: Pair the Screen to New Account](#Step2)

[L9] 

[L10] To move screens, devices from one account to another account, you will need to:

[L11] 

[L12] * Get the Pairing Code

[L13] * Remove device from old account

[L14] * Pair device to new account

[L15] 

[L16] Let's get started.

[L17] 

[L18] ---

[L19] 

[L20] ## Step 1: Unpair the Screen from Current Account

[L21] 

[L22] Unpairing the screen will be slightly different if you have device access, or if you want to unpair the screen via the Web Portal.

[L23] 

[L24] Click [**here**](#Portal) if you want to use the web portal, which can let you unpair and repair remotely. For use with the device, read on.

[L25] 

[L26] ### Using OptiSigns Player (With Device Access)

[L27] 

[L28] If you have access to the device, simply open the┬Ā**Hamburger menu** on the OptiSigns Player and hit **Unpair**.

[L29] 

[L30] ![optisigns player unpair option](https://support.optisigns.com/hc/article_attachments/38483285075347)

[L31] 

[L32] This will Unpair the device and cause the Pairing Code to display:

[L33] 

[L34] ![optisigns player pairing code](https://support.optisigns.com/hc/article_attachments/38483285082259)

[L35] 

[L36] Now, move on to [**step two**](#Step2).

[L37] 

[L38] ### Using OptiSigns Web Portal (Remote Access)

[L39] 

[L40] If you do not have access to your screen or device, or wish to move screens and accounts remotely, you'll need to perform a few additional steps.

[L41] 

[L42] First, go to the OptiSigns Web Portal. Log in to the account with the screen you wish to move. Find it on the **Screens** tab, then hit **Edit Screen ŌåÆ Advanced ŌåÆ More**.

[L43] 

[L44] ![edit screen advanced more](https://support.optisigns.com/hc/article_attachments/27458240678419)

[L45] 

[L46] Click the **Info (i)** button

[L47] 

[L48] ![info button edit screen](https://support.optisigns.com/hc/article_attachments/27458261252499)

[L49] 

[L50] There, you can find the Pairing Code inside the data:

[L51] 

[L52] ![pairing code data](https://support.optisigns.com/hc/article_attachments/38483257565971)

[L53] 

[L54] Write this pairing code down or copy/paste it and do not lose it, or else you will not be able to perform the next step.

[L55] 

[L56] Next, remove the device from the current account by backing out of this menu, clicking the **Three Dots**, then hitting┬Ā**Remove**:

[L57] 

[L58] ![web portal remove screen](https://support.optisigns.com/hc/article_attachments/38483257569427)

[L59] 

[L60] This will remove the device from your current account.

[L61] 

[L62] ---

[L63] 

[L64] ## Step 2: Pair the Screen to New Account

[L65] 

[L66] Now, log in to your other account. Navigate to the **Screens**tab, then hit **Add Screen**.

[L67] 

[L68] ![web portal add screen](https://support.optisigns.com/hc/article_attachments/38483257575315)

[L69] 

[L70] Type in the Pairing Code you obtained in Step 1, then hit **Pair**:

[L71] 

[L72] ![optisigns web portal pair screen menu](https://support.optisigns.com/hc/article_attachments/38483257576595)

[L73] 

[L74] Your screen will now be linked to this account.

[L75]
