[L1] # How to Connect to 802.1x Enterprise WiFi on Raspberry Pi

[L2] 

[L3] When you are trying to connect to 802.1x WiFi with WPA enterprise on Raspberry Pi,¬† you will see the WiFi access point is greyed out on Raspberry Pi, like below. It is because Raspberry Pi is using the simple network service on the GUI, which doesn't support enterprise WiFi.¬† However, Raspberry Pi is supporting 802.1x enterprise WiFi. And the simplest way to setup the connection to 802.1x enterprise WiFi on Raspberry Pi is to use standard Linux network manager to replace the simple network service. In this tutorial, we will walk through how to set it up.¬†¬†

[L4] 

[L5] ![mceclip0.png](https://support.optisigns.com/hc/article_attachments/4411750345875)

[L6] 

[L7] **Open the terminal window on Raspberry Pi, and run below commands to get the standard Linux network manager installed.**

[L8] 

[L9] ¬†

[L10] 

[L11] sudo apt install network-manager network-manager-gnome  

[L12] sudo systemctl disable --now dhcpcd  

[L13] sudo systemctl enable --now network-manager

[L14] 

[L15] sudo reboot

[L16] 

[L17] **Explanation:**

[L18] 

[L19] 1st line is to install the Linux network manager

[L20] 

[L21] 2nd line is to disable the DHCP, because DHCP will not work with the network manager

[L22] 

[L23] 3rd line is to enable the network manager service

[L24] 

[L25] Last line is to reboot the device.

[L26] 

[L27] **After rebooting the device, you will be able to see a new network icon, click on the icon, you will not be able to connect to the 802.1x enterprise WiFi.**

[L28] 

[L29] ![mceclip1.png](https://support.optisigns.com/hc/article_attachments/4411743714067)

[L30] 

[L31] ¬†

[L32] 

[L33] **Then fill in the information as you normally do with 802.1x enterprise WiFi connection, and you will now be able to connect to the network.**

[L34] 

[L35] ![mceclip2.png](https://support.optisigns.com/hc/article_attachments/4411743744787)

[L36] 

[L37] ¬†

[L38] 

[L39] **That's it! You're ready to go.**

[L40] 

[L41] **Note:** To download and install the network manager, you will need internet connection first. Please consider have it connected to a wired network or another normal WiFi network first to get it setup.
