[L1] # How to Create and Use Nested Playlists

[L2] 

[L3] ## Introduction

[L4] 

[L5] Sometimes, you need to re-use a playlist inside another playlist. This feature is called a nested playlist or sub-playlist.

[L6] 

[L7] |  |

[L8] | --- |

[L9] | **NOTE** |

[L10] | Nested Playlists are available to┬Ā**[Standard plan](https://www.optisigns.com/pricing)**users and above. |

[L11] 

[L12] For example, there may be a need for a common or global playlist that you want for all of your screens but require there to be some exceptions for regional-specific content to also be shown. Nested Playlists allow you to be able to manage that in an organized manner.

[L13] 

[L14] This article assumes you already know how to [create and use Playlists](https://support.optisigns.com/hc/en-us/articles/28295104605843) with OptiSigns.   

[L15] If you're not sure, check out that article first.

[L16] 

[L17] ## Let's dive in:

[L18] 

[L19] In this example, we have a Global Playlist that has corporate-level content, and we have Regional playlist where it needs to play the Global Playlist content plus some localized content.

[L20] 

[L21] To add an existing Global Playlist into the new Regional playlist, you can navigate to the Playlist tab.

[L22] 

[L23] Click "Add Playlist" button

[L24] 

[L25] ![mceclip2.png](https://support.optisigns.com/hc/article_attachments/360058309474)

[L26] 

[L27] Select the playlist you want to add as nested playlist.  

[L28] In this case it's the Global Playlist.  

[L29] Then click Add.

[L30] 

[L31] ![mceclip1.png](https://support.optisigns.com/hc/article_attachments/360058309374)

[L32] 

[L33] Now you can see your Global Playlist inside your Regional playlist.  

[L34] You can move the order of the Global Playlist just like any other playlist items.

[L35] 

[L36] ![mceclip3.png](https://support.optisigns.com/hc/article_attachments/360059221433)

[L37] 

[L38] That's it, you can now assign this Regional playlist to any screens and it will play contents in the order as arranged.

[L39] 

[L40] Just like other playlist items, you can set limit duration of the Child playlist in the Parent playlist (to control how long this item will be played).  

[L41] If the child playlist duration is shorter than this duration, it will be repeated till this duration is done.  

[L42] If the child playlist duration is longer than this duration, it will get cut off at this duration.

[L43] 

[L44] ![mceclip0.png](https://support.optisigns.com/hc/article_attachments/360065122574)

[L45] 

[L46] **Note:**About playlist transition.

[L47] 

[L48] If you set specific transition for playlist items in Global, this transition setting will remain.
