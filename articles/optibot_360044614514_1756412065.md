[L1] # There's no sounds output for my Raspberry Pi

[L2] 

[L3] Make sure you setting audio out to the right output.

[L4] 

[L5] **Solution 1:**

[L6] 

[L7] To change audio output, right click on the audio icon and select the output.

[L8] 

[L9] ![mceclip1.png](https://support.optisigns.com/hc/article_attachments/360061945813)

[L10] 

[L11] If you are using Raspberry Pi 4, as of 3/12/2020 there's some know issues output audio to both HDMI ports. Use the port next to the power input, it works.

[L12] 

[L13] **Solution 2:**

[L14] 

[L15] Open the **Terminal,** run command line, use this command to switch the audio output to HDMI:

[L16] 

[L17] ```

[L18] amixer cset numid=3 2

[L19] ```

[L20] 

[L21] If output is set to `2`, which is HDMI.  Output to `1` is analogue (headphone jack). Default setting is `0` is automatic.

[L22] 

[L23] **Solution 3:**

[L24] 

[L25] Open the **Terminal,** run command line.

[L26] 

[L27] ```

[L28] sudo raspi-config

[L29] ```

[L30] 

[L31] ![mceclip6.png](https://support.optisigns.com/hc/article_attachments/8571021810579)

[L32] 

[L33] Select **System Options**.

[L34] 

[L35] ![mceclip1.png](https://support.optisigns.com/hc/article_attachments/8570979748499)

[L36] 

[L37] Select **Audio**.

[L38] 

[L39] ![mceclip2.png](https://support.optisigns.com/hc/article_attachments/8570950798611)

[L40] 

[L41] Select **0HDMI**

[L42] 

[L43] ![mceclip3.png](https://support.optisigns.com/hc/article_attachments/8570985894931)

[L44] 

[L45] Then select **Finish**

[L46] 

[L47] ![mceclip4.png](https://support.optisigns.com/hc/article_attachments/8570984227731)

[L48] 

[L49] After that, run reboot in the command to reboot your Raspberry Pi

[L50] 

[L51] ![mceclip5.png](https://support.optisigns.com/hc/article_attachments/8570988279571)
