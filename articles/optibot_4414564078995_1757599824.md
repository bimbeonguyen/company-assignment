[L1] # Error Handling

[L2] 

[L3] Whenever our server encounters errors while processing a GraphQL operation, it will include an error object in the response. The error object has an error array that contains each error occured. Each error in the array has a message field that contains the error message, and an extensions field that provides additional useful information, including an error code.
