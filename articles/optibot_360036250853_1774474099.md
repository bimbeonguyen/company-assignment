[L27] ## Creating a Microsoft Outlook Calendar App

[L28] 

[L29] To display Microsoft Outlook Calendar onscreen, you'll need to create an app within OptiSigns.

[L30] 

[L31] First, go to the [OptiSigns Portal](http://app.optisigns.com/). Go to **Files/Assets ‚Üí Add Asset ‚Üí Apps.**

[L32] 

[L33] ![optisigns files assets apps](https://support.optisigns.com/hc/article_attachments/50265323153171)

[L34] 

[L35] Select **Outlook Calendar**.

[L36] 

[L37] ![optisigns find outlook calendar app](https://support.optisigns.com/hc/article_attachments/50265323157395)

[L38] 

[L39] The below screen should appear:

[L40] 

[L41] ![optisigns outlook calendar smallapp](https://support.optisigns.com/hc/article_attachments/50265323159059)

[L42] 

[L43] Here, you have a few options.

[L44] 

[L45] If you have used a Microsoft account on the OptiSigns platform, you can select it from the dropdown. If you have not, you'll need to hit **Login with Work Account**, enter in your Microsoft account information, and grant OptiSigns the necessary permissions. If you're using a Personal Account, click **Login with Personal Account**.

[L46] 

[L47] Then, hit **Next** within OptiSigns.

[L48] 

[L49] The below screen will then appear (with nothing filled in), along with a Preview:

[L50] 

[L51] ![optisigns outlook calendar app setup](https://support.optisigns.com/hc/article_attachments/50265338911507)

[L52] 

[L53] * **Name -** Name of your calendar that you wish to see it labeled as within the OptiSigns platform. This name is internal to you only and ***will not*** be displayed on your screens.

[L54] * **Calendars -**¬† The dropdown will show all the calendars that are within the account. These can be calendars you created or subscribed to.

[L55] * **Title -** Title of the calendar. This ***will*** be displayed onscreen.

[L56] * **Display Mode -** Select the view of the calendar

[L57] 	+ **Schedule -** Linear listing of every event in succession by time and date.

[L58] 	+ **Day -** Listing the current day's event activity.

[L59] 	+ **Week -** Listing the current week's event activity

[L60] 	+ **Month -** Listing the current month's event activity

[L61] * **Only Show Ongoing and Upcoming Events -**

[L62] 

[L63] You may also switch between **Landscape** and **Portrait** for your Preview, depending on the orientation of the display you wish to display the app on.

[L64] 

[L65] ![optisigns outlook calendar portrait display](https://support.optisigns.com/hc/article_attachments/50265323162259)

[L66] 

[L67] #### Theme Settings

[L68] 

[L69] ![optisigns outlook calendar theme settings](https://support.optisigns.com/hc/article_attachments/50265338917907)

[L70] 

[L71] Toggle this open or closed. These settings will be reflected in the Preview on the right.

[L72] 

[L73] * **Hide Background Image -** Removes the Background Image from the app.

[L74] * **Background Image -** Choose between the Default image, or a Custom Background image. If Custom is chosen, a new option will appear:

[L75] * ![optisigns outlook calendar background image options](https://support.optisigns.com/hc/article_attachments/50265466771219)

[L76] * **Choose Photo -** Select the photo you wish to use as a Background Image. This must be an existing asset within OptiSigns.

[L77] * **Theme -** Choose between Light and Dark theme for your displays.

[L78] * **Text Color -** Change the color of your calendar's text.

[L79] * **Use Your Calendar Event Color -** Check to keep the existing event colors from your calendar. Uncheck if you want to change them.

[L80] 

[L81] ![optisigns outlook calendar use calendar event color options](https://support.optisigns.com/hc/article_attachments/50265323168787)

[L82] 

[L83] * **Event Background Color -** Appears when "Use Your Calendar Event Color" is unchecked. This lets you change the color of how the event appears.

[L84] * **Event Text Color -** Appears when "Use Your Calendar Event Color" is unchecked. This lets you change the color of how the text on the event appears.

[L85] * **Font Size -** Change the size of the font on your Microsoft Outlook Calendar app.

[L86] 

[L87] #### Advanced Options

[L88] 

[L89] ![optisigns outlook calendar app advanced options](https://support.optisigns.com/hc/article_attachments/50265323170195)

[L90] 

[L91] * **Time Range** **-** Choose the range of time you want to display. By default, it is set to **Full Day**, and you can choose **8AM - 5PM**, **9AM - 5PM**, or **Custom**. This will let you fine-tune the times you wish to actually show your calendar.

[L92] * **Slot spacing -** Changes the spacing of the slots within the calendar.

[L93] * **Show Location -** Show the location of your calendar's event.

[L94] * **Auto Scroll -** By default, the calendar will focus the display of events on current local time or your set time zone time. This can be less helpful if you anticipate viewers to be able to see all event activities of your calendar.¬†  

[L95] Toggle the auto scroll pans the schedule so that all times are viewable.

[L96] * **Time Format -** Change between 12 hour and 24 hour clocks.

[L97] * **Weather -** Entering location information allows weather information to be overlaid on your calendar.

[L98] * **Timezone -** Change the timezone of the meeting. This defaults to the used device's timezone, but can be specified here if needed.

[L99] * **Language -** Choose between a variety of language options for the display.

[L100] 

[L101] Once you've set up the app, it's ready to be deployed.

[L105] ## Deploying a Microsoft Outlook Calendar App

[L106] 

[L107] You can deploy your new Microsoft Outlook Calendar app as an individual asset, or as part of a [Split Screen](https://support.optisigns.com/hc/en-us/articles/360026559573-How-to-Create-and-Use-the-Split-Screen-App).

[L108] 

[L109] To get your new Microsoft Outlook Calendar asset to a screen, go to the **Screens** tab, then click the screen you want to assign it to.

[L110] 

[L111] ![optisigns screens edit](https://support.optisigns.com/hc/article_attachments/50265338926099)

[L112] 

[L113] This brings up the **Edit Screen** tab:

[L114] 

[L115] ![optisigns edit screen popup](https://support.optisigns.com/hc/article_attachments/50265338927635)

[L116] 

[L117] Here, select **Asset** under Content type, then hit **Browse** next to Selected Asset (if needed).

[L118] 

[L119] Then, select your created Microsoft Outlook Calendar asset:

[L120] 

[L121] ![optisigns select asset outlook calendar](https://support.optisigns.com/hc/article_attachments/50265338930963)

[L122] 

[L123] Now hit **Save**. Your Microsoft Outlook Calendar asset will now display on screen.¬†

[L134] ## Frequently Asked Questions

[L135] 

[L136] #### Why does using Outlook or Office 365 Calendar require giving OptiSigns Admin permission to use?

[L137] 

[L138] OptiSigns uses Microsoft APIs for integration. In order for our integrations to work, the integration has to be approved by an administrator. This is the same across all integrations using Microsoft APIs.

[L139] 

[L140] This administrator access is only needed for first time access. Once the OptiSigns app is approved for use, other users can use OptiSigns directly.

[L141] 

[L142] #### I can't find the calendar I want to display. Why not?

[L143] 

[L144] A limitation of the integration is that OptiSigns can only display what is tied to a given Microsoft account. This means that any calendars shared with that account or present in Microsoft Rooms cannot be displayed.

[L145] 

[L146] To fix this, you can change the calendar directory from Rooms to a personal one. First, find the calendar you wish to display, then hit the¬†**3-dot icon**and navigate to¬†**Remove**:

[L147] 

[L148] ![image (24)(1).png](https://support.optisigns.com/hc/article_attachments/42117346771219)

[L149] 

[L150] Next, hit¬†**Add Calendar**. Find the calendar from the directory you wish to display, then add it to¬†**My Calendars:**

[L151] 

[L152] ![image (25)(1).png](https://support.optisigns.com/hc/article_attachments/42117351512467)

[L153] 

[L154] Now, the appropriate calendar should display once it's been recreated in OptiSigns.
