[L1] # How to use a PDF with OptiSigns

[L2] 

[L3] There are 3 ways you can use a PDF file:

[L4] 

[L5] * Assign it directly to a screen as an asset

[L6] * Put it in a playlist

[L7] * Put it in a schedule

[L8] 

[L9] ### If you're using a PDF directly as an asset on the screen:

[L10] 

[L11] You'll need to enter a duration (10 pages, if you enter 200s, each page will be 20s)![](/attachments/token/IZoaFRKd7k98GJXJ7VxJtvvd9/?name=inline-118732892.png)​

[L12] 

[L13] ### If you're using PDF file in a Playlist:

[L14] 

[L15] The item duration is the duration for the document (i.e. 200s and the doc has 10 pages, each page will be displayed for 20s).

[L16] 

[L17] ![](/attachments/token/NElNaFXL3qBlZyWjNkmmmRdd2/?name=inline-1360819140.png)​

[L18] 

[L19]  

[L20] 

[L21] ### If you are using it in a schedule and assign the PDF as a scheduled asset:

[L22] 

[L23] You'll need to enter a duration, like a single asset (10 pages, if you enter 200s, each page will be 10s).

[L24] 

[L25] ![](/attachments/token/sLid2HpAXc2sXXP6VpPwq4Be1/?name=inline35816297.png)​

[L26] 

[L27] ### Troubleshoot:

[L28] 

[L29] There are many different versions and ways that a PDF can be generated. So, sometimes you will experience issues with PDF playback.

[L30] 

[L31] In general, PDFs generated by Google Docs, Slides, Microsoft Office applications, or Adobe applications will play fine.

[L32] 

[L33] Sometimes, PDFs that are too large can have issues. In that case, you can convert the PDF to Images and use it that way.
