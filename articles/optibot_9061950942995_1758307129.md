[L1] # Using RS-232 to Schedule TV Power On/Off or other commands

[L2] 

[L3] With OptiSigns, you can schedule TV Power On/Off using the advanced schedule feature. There are 2 ways you can do it with OptiSigns depending on what devices you are using.

[L4] 

[L5] * If you are using a consumer-grade TV and an OptiStick, the TV Power On/Off is achieved through HDMI-CEC. See our **[Operational Schedule](https://support.optisigns.com/hc/en-us/articles/28598173096723-How-To-Create-and-Use-Operational-Schedules-HDMI-CEC-RS-232)**article for more information.

[L6] * If you are using a commercial-grade display, RS232 will be the option for you.

[L7] 

[L8] In this article, we will walk through how to set up the external communication for RS-232 and use it to control your commercial display power on/off. It consists of 3 steps at a high level.

[L9] 

[L10] 1. Set up RS-232 connection for serial communication.

[L11] 2. Configure the RS-232 command for your display and set up the template

[L12] 3. Create the Power On/Off Schedule

[L13] 

[L14] The same procedures can be used to send other RS 232 compliant commands as well.

[L15] 

[L16] Supported devices/platforms: Windows, Linux, Raspberry Pi, OptiStick, and OptiSigns Pro/ProMax players.

[L17] 

[L18] **1. Create RS-232 connection**

[L19] 

[L20] Go to the admin console and expand Advanced, click on the link for external communication. You can also use the link below.

[L21] 

[L22] <https://app.optisigns.com/app/s/external-coms>

[L23] 

[L24] Click "Add New" button, to create a new RS-232 connection.¬†

[L25] 

[L26] ![chrome_ThjlLpoiH0.png](https://support.optisigns.com/hc/article_attachments/32323154940819)

[L27] 

[L28] In the connection creation window, you can give a name to the connection and define the serial port to use, Baud Rate, Data bits, Stop bits, and Parity. This info is determined by the commercial display used, you should be able to get it from your TV user manual.¬† Flow control is normally not needed in this case, you can just leave it blank.

[L29] 

[L30] ![chrome_972ixLupAY.png](https://support.optisigns.com/hc/article_attachments/32323154949651)

[L31] 

[L32] Click "Save", once the configuration is complete.¬†

[L33] 

[L34] To know which serial port is used, you will need to check it on your device. Using Windows as an example, you can find it in the device manager, it will list out the COM port available. On Linux/Raspberry Pi, the serial port is normally "/dev/ttyS0" or "/dev/ttyUSB0" if a USB serial adapter is used. On OptiSigns pre-configured Android Stick, you can use "USB0" as the serial port.¬†

[L35] 

[L36] ![mceclip2.png](https://support.optisigns.com/hc/article_attachments/9062308259731)

[L37] 

[L38] ¬†

[L39] 

[L40] **2. Configure RS-232 command and set up the template**

[L41] 

[L42] Go to the "Commands" tab, and click the "Add New" button.

[L43] 

[L44] ![chrome_kVTicFpYv5.png](https://support.optisigns.com/hc/article_attachments/32323154954003)

[L45] 

[L46] Set the RS-232 command of your commercial display in the pop-up window. You can choose "asc ii" or "hex" encoding depending on your TV, and the end of line value can be set here accordingly as well.¬†

[L47] 

[L48] You will need to create a command for Power On, and a command for Power Off.

[L49] 

[L50] ![chrome_QGt8AmZbeI.png](https://support.optisigns.com/hc/article_attachments/32323291436947)

[L51] 

[L52] Once commands are created, go to the "Templates" tab, this is where you map the commands to the Power On/Off action. You only need to create a template if the commands are different. For example, if you are only using Samsung commercial TVs, which all have the same commands for power on/off, then you only need to create one template for it, it can be used on all the Samsung TVs you have deployed.¬† ¬†¬†

[L53] 

[L54] ¬†

[L55] 

[L56] ![chrome_0RqFJkgyu4.png](https://support.optisigns.com/hc/article_attachments/32323151390739)

[L57] 

[L58] Map the command to the actions. The actions will be used in the advanced schedule. Save it after the mapping is complete, then the template is ready for use.

[L59] 

[L60] ![mceclip7.png](https://support.optisigns.com/hc/article_attachments/9062485175059)

[L61] 

[L62] **3. Set up Power On/Off schedule**

[L63] 

[L64] Once the RS-232 configuration is complete, you will need to set up an Operational Schedule.

[L65] 

[L66] Go to the Edit Screens, then go to the **Operational Schedule**section. Hit¬†**New**.

[L67] 

[L68] ![](https://support.optisigns.com/hc/article_attachments/40414890419859)

[L69] 

[L70] You'll see the Operational Schedule screen. Here, click anywhere you'd like to schedule a Power On/Off.

[L71] 

[L72] ![](https://support.optisigns.com/hc/article_attachments/40414890422803)

[L73] 

[L74] Once you've done this, you'll see this menu pop up on the side:

[L75] 

[L76] ![](https://support.optisigns.com/hc/article_attachments/40414884380947)

[L77] 

[L78] Here, the relevant piece is **Power**. Clicking this will show a drop-down:

[L79] 

[L80] ![](https://support.optisigns.com/hc/article_attachments/40414890424723)

[L81] 

[L82] To power your screen on or off using RS-232, there are two relevant options:

[L83] 

[L84] * **To Power Screen ON -** Choose either **On**or **RS-232 - On**. The first option will make use of RS-232 first, then HDMI-CEC as a fallback. The second will only utilize RS-232.

[L85] * **To Power Screen OFF -**Choose either **Off**or¬†**RS-232 - Off**. The first option will make use of RS-232 first, then HDMI-CEC as a fallback. The second will only utilize RS-232.

[L86] 

[L87] That's all! Now you have learned how to schedule TV power on/off using RS-232.
