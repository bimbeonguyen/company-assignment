[L1] # Adjust the GPU memory on the Raspberry Pi

[L2] 

[L3] Sometimes, videos won't run smoothly on your Raspberry Pi. Adjusting the GPU memory on your Raspberry Pi can help solve this issue.

[L4] 

[L5] In this guide, we'll walk you through how to adjust the GPU memory on your Raspberry Pi.

[L6] 

[L7] ## Let's jump in and get started:

[L8] 

[L9] 1. Open the terminal on the Raspberry Pi.

[L10] 

[L11] ![1.png](https://support.optisigns.com/hc/article_attachments/31634726857107)

[L12] 

[L13] 2. Type **sudo raspi-config** in the terminal.

[L14] 

[L15] ![2.png](https://support.optisigns.com/hc/article_attachments/31634726859539)

[L16] 

[L17] 3. Select **Performance Options.**

[L18] 

[L19] **![3.png](https://support.optisigns.com/hc/article_attachments/31634705160723)**

[L20] 

[L21] 4. Select **GPU Memory**.

[L22] 

[L23] **![4.png](https://support.optisigns.com/hc/article_attachments/31634726863635)**

[L24] 

[L25] 5. Update the number to **256**, then select **OK**.

[L26] 

[L27] **![5.png](https://support.optisigns.com/hc/article_attachments/31634705167123)**

[L28] 

[L29] 6. Select **Finish**.

[L30] 

[L31] ![6.png](https://support.optisigns.com/hc/article_attachments/31634705170963)

[L32] 

[L33] 7. Lastly, select **Yes** to reboot your Raspberry Pi.

[L34] 

[L35] ![7.png](https://support.optisigns.com/hc/article_attachments/31634726882067)

[L36] 

[L37] ### That's it!

[L38] 

[L39] The GPU memory is now set to 256 MB on your Raspberry Pi.
