[L1] # How to manually install & activate OptiSigns AI Add-on on Linux or Raspberry Pi

[L2] 

[L3] OptiSigns AI Add-on is a background service running on your Linux, Raspberry Pi device to use a Camera to detect & count people passing by.

[L4] 

[L5] If you have not tried to automatically activate this feature, please try it first by following [this guide](https://support.optisigns.com/hc/en-us/articles/27690296225555).

[L6] 

[L7] The add-on is a ~200MB app, so it will take several minutes to download and activate on your device, so please be patient after auto-activation before you try manual activation.

[L8] 

[L9] If auto-activation fails, then you can follow these steps to manually install & activate AI detection add-on on your Linux or Raspberry Pi devices.

[L10] 

[L11] ┬Ā

[L12] 

[L13] **Before you start, please ensure that you have:**

[L14] 

[L15] * Your device has OptiSigns Digital Signage player version 4.2.10 or newer installed. You can download the latest version [here](https://www.optisigns.com/download).

[L16] * There's a network connection to your device

[L17] * You have enough authorization, and privileges to install apps, and background services on your device

[L18] * optisigns-ai-detection runs on any Raspberry Pi OS, or any major Linux distro such as Ubuntu.

[L19] 

[L20] ## **How to manually install optisigns-ai-detection**

[L21] 

[L22] Make sure you have a camera installed on your system.  

[L23] Open Terminal.

[L24] 

[L25] On Raspberry Pi run:

[L26] 

[L27] ```

[L28] curl -s https://links.optisigns.com/ai-add-on-rpi-install -L | sh

[L29] ```

[L30] 

[L31] On Linux run:

[L32] 

[L33] ```

[L34] curl -s https://links.optisigns.com/ai-add-on-linux-install -L | sh

[L35] ```

[L36] 

[L37] This will download a script and run the installation.

[L38] 

[L39] ![mceclip0.png](https://support.optisigns.com/hc/article_attachments/360099273994)

[L40] 

[L41] After you see "Starting OptiSigns AI Detection Service ..." you are done!

[L42] 

[L43] You can go back to [app.optisigns.com](https://app.optisigns.com/app/screenManagement) to use the AI feature now

[L44] 

[L45] ## **Further technical detail for optisigns-ai-detection**

[L46] 

[L47] The binary executable is downloaded and installed to ~/bin/optisigns-ai-detection

[L48] 

[L49] It will be started as a background service and will be automatically started on system start-up.

[L50] 

[L51] This will ensure the AI detection add-on is independent of OptiSigns Digital Signage Player and still running in case the Player has an issue or you need to reinstall it.

[L52] 

[L53] You can manually execute the optisigns-ai-detection executable:

[L54] 

[L55] ```

[L56] <path>/optisigns-ai-detection [arg]  

[L57] install : to install ai detection agent  

[L58] remove : to remove/uninstall ai detection agent  

[L59] start or restart : to start ai detection agent service  

[L60] stop : to stop ai detection agent service  

[L61] version or -V : to print ai detection agent version  

[L62] help : to print this instructions  

[L63] run : to run the ai detection agent in console or terminal or service  

[L64] show or -s: show camera feed

[L65] ```

[L66] 

[L67] ## **How to remove optisigns-ai-detection:**

[L68] 

[L69] Open Terminal and run:

[L70] 

[L71] ```

[L72] ~/bin/optisigns-ai-detection remove

[L73] ```
