[L1] # Your Templates

[L2] 

[L3] * [Saving and Using Custom Templates](#Templates)

[L4] * [Saving and Using Custom Components](#Components)

[L5] 

[L6] In Designer, the **Your Temp** (short for “Your Templates”) tab allows you to save and reuse your created templates across multiple designs.

[L7] 

[L8] For more general information on the Designer app, see our[**Getting Started in Designer**](https://support.optisigns.com/hc/en-us/articles/42087942047379-Getting-Started-with-Designer)guide.

[L9] 

[L10] This feature can be accessed from the Side Menu within Designer:

[L11] 

[L12] ![](https://support.optisigns.com/hc/article_attachments/42830684533779)

[L13] 

[L14] Within this tab, there are two options: **Templates** and **Components.**

[L15] 

[L16] * **Templates** allows you to save the current design as a Custom Template for use in later designs. This saves the ***entire design*** as a template.

[L17] * **Components** allows you to save a selected component (be that text, image, shape, widget, etc.) as a custom component for use in later designs. This saves ***just the selected component*** as a template.

[L18] 

[L19] Below, we’ll cover how to save and store these custom templates and give examples of how they can be used.

[L20] 

[L21] ---

[L22] 

[L23] ## Saving and Using Custom Templates

[L24] 

[L25] The **Templates** tab has one option when you first open it: **Add Current Design as Template**. Nothing needs to be selected in order to use this.

[L26] 

[L27] ![](https://support.optisigns.com/hc/article_attachments/42830652782483)

[L28] 

[L29] Clicking this will bring a popup asking you to confirm your choice. Click **Add**. This saves whatever you have on the current design as a Custom Template, shared across your team.

[L30] 

[L31] ![custom template added](https://support.optisigns.com/hc/article_attachments/42830684537875)

[L32] 

[L33] |  |

[L34] | --- |

[L35] | **NOTE** |

[L36] | The Custom Template may appear as a black box when it is first added to the design. However, itwill still function normally. Saving the design and relaunching it should make it appear.  |

[L37] 

[L38] To use this template, simply click on it from the Side Menu:

[L39] 

[L40] ![custom template being added](https://support.optisigns.com/hc/article_attachments/42830684540179)  

[L41] This will be present across any Designer instance.

[L42] 

[L43] To delete a Custom Template, hover over it and click the trash can that appears.

[L44] 

[L45] ![custom template how to delete](https://support.optisigns.com/hc/article_attachments/42830652788627)

[L48] 

[L49] ## Saving and Using Custom Components

[L50] 

[L51] The **Components** tab has one option when you first open it: **Add Current Selection as Component**. A component on the current design needs to be selected for this to be used.

[L52] 

[L53] ![component tab add current selection as component button](https://support.optisigns.com/hc/article_attachments/42830652789907)

[L54] 

[L55] Clicking this will bring up a popup asking you to confirm your choice. Click **Add**. This saves the selected component as a Custom Component, shared across your team.

[L56] 

[L57] ![custom component added](https://support.optisigns.com/hc/article_attachments/42830652796563)

[L58] 

[L59] |  |

[L60] | --- |

[L61] | **NOTE** |

[L62] | The Custom Component may appear as a black box when it is first added to the design. However, itwill still function normally. Saving the design and relaunching it should make it appear. |

[L63] 

[L64] To use this component, simply click on it from the Side Menu:

[L65] 

[L66] ![add component to design](https://support.optisigns.com/hc/article_attachments/42830652799507)

[L67] 

[L68] This component can then be moved around and placed as needed. It will be present across any Designer instance.

[L69] 

[L70] To delete a Custom Component, hover over it and click the trash can:

[L71] 

[L72] ![custom component how to delete](https://support.optisigns.com/hc/article_attachments/42830684546067)
