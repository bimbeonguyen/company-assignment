[L1] # Clean install OptiSigns on Raspberry Pi/Linux

[L2] 

[L3] To completely clean out old installation of OptiSigns on Linux or Raspberry Pi

[L4] 

[L5] Please run:

[L6] 

[L7] ```

[L8] rm -rf ~/.config/OptiSigns  

[L9] rm ~/.config/autostart/'OptiSigns Digital Signage.desktop'

[L10] ```

[L11] 

[L12] Also delete the long string text on this ~/.config folder

[L13] 

[L14] Then install the new AppImage download from <https://www.optisigns.com/download>
