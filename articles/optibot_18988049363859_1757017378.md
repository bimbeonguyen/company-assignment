[L1] # Push Contents to your Screens

[L2] 

[L3] When managing multiple screens with OptiSigns, you have two primary methods to push your content to the screens. Below is a detailed guide to help you choose the best method and execute it effectively.

[L4] 

[L5] * [Method 1: Edit Screen and Assign Content](#1)

[L6] * [Method 2: "Push to Screens" Button](#2)

[L7] 

[L8] Let's get started!

[L9] 

[L10] ---

[L11] 

[L12] ## Method 1: Edit screen and assign the content

[L13] 

[L14] This method is more suitable if you are assigning content on specific screens and you want it to go live immediately. You can also select a list of screens and do a group edit.┬Ā

[L15] 

[L16] ### To Edit One Screen:

[L17] 

[L18] 1. Go to the [Screens Management](https://app.optisigns.com/app/screenManagement) page, and find your desired screen

[L19] 2. Click **Edit** ![](https://support.optisigns.com/hc/article_attachments/31406869798803)

[L20] 3. In the Edit Screen pop-up window, select **Type** and choose either Asset, Playlist or Schedule

[L21] 4. Click on **Selected Asset** and choose your desired content from the pop-up window

[L22] 5. Click **Save**

[L23] 

[L24] ![mceclip1 (1).png](https://support.optisigns.com/hc/article_attachments/31406895597203)

[L25] 

[L26] ### To Edit Multiple Screens:

[L27] 

[L28] 1. Go to the [Screens Management](https://app.optisigns.com/app/screenManagement) page

[L29] 2. Select each checkmark box next to Screens

[L30] 3. Click **Edit**

[L31] 4. In the Edit Group of Screen pop-up window, select **Type** and choose either Asset, Playlist or Schedule

[L32] 5. Click on **Selected Asset** and choose your desired content from the pop-up window

[L33] 6. Click **Save**

[L34] 

[L35] ![](https://support.optisigns.com/hc/article_attachments/31406869805715)

[L39] ## Method 2: "Push to Screens" button

[L40] 

[L41] This method allows the user to apply more conditions when pushing content to screens. You can access the Push to Screens button in multiple areas:

[L42] 

[L43] * **Files/Assets:** Go to the Files/Assets page  ŌåÆ Click on the 3 Vertical Dots ŌåÆ Select the Push To Screens

[L44] * **Playlists:** Go to the Playlist page ŌåÆ Select the Push to Screens Button

[L45] * **Schedules:** Go to the Schedules page ŌåÆ Select the Push to Screens button

[L46] * **Split Screen app:** Open a Split Screen app ŌåÆ Select the Push to Screens in the top right corner

[L47] * **Designer:** Open the Designer App ŌåÆ Select the Push to Screen button in the top right corner

[L48] 

[L49] #### When pushing content to the screen, there are multiple options of **when** you want the content to go live:

[L50] 

[L51] ![](https://support.optisigns.com/hc/article_attachments/31407460275731)

[L52] 

[L53] * **Push:** Assign and push the content to screens immediately

[L54] * **Schedule Changes:** Select a time for the content to start playing. Optional, set an expiration time, if you'd like the content to stop playing at some point or revert to the last played content.

[L55] * **Temp Takeover:** Temporarily take over the screens for a selected period of time before reverting back to the previously assigned content.

[L56] 

[L57] #### There are also multiple options on **where** to target the content to:

[L58] 

[L59] [![](https://support.optisigns.com/hc/article_attachments/31407460277651)](https://support.optisigns.com/hc/en-us/articles/20879903340947)

[L60] 

[L61] * **Screens:** Select the screen(s) you want to target

[L62] * **Tags:**If your screens have assigned tags, select those tags and any screen with those tags will be targeted.

[L63] * **Tag Rules:** Allows you to define more complex conditions when selecting target screens, which you can learn more [here.](https://support.optisigns.com/hc/en-us/articles/20879903340947)

[L67] ### What Happens if Your Screen Already has an Assigned Playlist?

[L68] 

[L69] When the target screens are currently running a playlist, when you push the contents to the screen, you will be given the option to either include the asset in the currently running playlist or override the playlist with the asset. It will show "**Screens(s) using playlist**".┬Ā

[L70] 

[L71] ![](https://support.optisigns.com/hc/article_attachments/19209188203283)

[L72] 

[L73] * **Add this asset to the end of the current playlist(s) the screen(s)**: This will add the asset to the end of the current playlist. You can also set the duration (in seconds) to it.

[L74] * **Assign this asset to the selected screen(s)**: This will just assign this asset to the screen directly.
