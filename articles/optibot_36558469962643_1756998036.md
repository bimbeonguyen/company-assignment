[L1] # Subscription Function in GraphQL

[L2] 

[L3] Subscriptions are a query type that allows you to be notified when any changes are made to a device, asset, or playlist.В 

[L4] 

[L5] An example might be that you want to be notified of any changes made to your device, be that changing the asset or playlist. This can be done entirely within GraphQL by using queries to find the information about a device, setting up a subscription, then using mutations to alter the device attributes.

[L6]
