[L18] ## Enabling Unknown Source Downloads

[L19] 

[L20] Newer versions of Android will need you to allow **Unknown Source** downloads in order to successfully install the OptiSigns app. In this case, you know the source - it's us! - but your Android device may not recognize it. This will not be needed on every Android device, but it's best to do it just to be safe.

[L21] 

[L22] |  |

[L23] | --- |

[L24] | **NOTE** |

[L25] | These instructions mainly apply to Android Oreo, Pie, 9.0, or later. For even more detailed instructions on Unknown Sources, [see how to install apps from unknown sources in Android](https://www.appaloosa.io/blog/guides/how-to-install-apps-from-unknown-sources-in-android). |

[L26] 

[L27] To allow Unknown Source downloads, go to **Settings ŌåÆ** **Device Preferences ŌåÆ Security & restrictions.**

[L28] 

[L29] ![android device preferences security & restrictions](https://support.optisigns.com/hc/article_attachments/36321849020179)

[L30] 

[L31] Here, hit **Unknown Sources**. Several apps will appear. If you plan to download OptiSigns via Chrome Web Browser, make sure it's **Enabled**. If via an External Source, enable that. This may also present as a File Manager. Whatever source it is, Enable it.

[L32] 

[L33] ![android unknown source apps](https://support.optisigns.com/hc/article_attachments/36321880532371)

[L34] 

[L35] In this example, we've enabled Chrome. Now we're ready to get OptiSigns on your device.

[L39] ## Installing APK via an External Device

[L40] 

[L41] These steps should be followed if you do not have convenient access to a **Web Browser** on your device, or unstable access to a network connection. If you have a Web Browser (but no Google Play store), we recommend installing [Via Web Browser](#WebBrowser).

[L42] 

[L43] ### Step 1: Download OptiSigns APK and Put on External Storage Device

[L44] 

[L45] On a computer, navigate to the [OptiSigns Download page](https://www.optisigns.com/download). Scroll down until you see the Android logo. This is where you will download the APK. Click anywhere in the box.

[L46] 

[L47] ![optisigns downloader android](https://support.optisigns.com/hc/article_attachments/36321880534675)

[L48] 

[L49] These options will pop up. Click **Download APK**.

[L50] 

[L51] ![optisigns download apk](https://support.optisigns.com/hc/article_attachments/36321849029395)

[L52] 

[L53] Now find the APK in your Downloads folder (or wherever it was downloaded to) and place it on your External Storage device.

[L54] 

[L55] ### Step 3: Attach External Storage to Device and Run APKInstaller

[L56] 

[L57] Plug your external storage device into the Android device. Find your pre-installed APKInstaller. This can usually be found in the **"More Apps"** section, indicated by multiple dots.

[L58] 

[L59] ![more apps section six dots](https://support.optisigns.com/hc/article_attachments/36321880540307)

[L60] 

[L61] Open your **APKInstaller**. The precise version of this may vary by device, but should exist in some form if your device lacks a web browser.

[L62] 

[L63] ![apkinstaller example](https://support.optisigns.com/hc/article_attachments/36321849033107)

[L64] 

[L65] Grant access to your external storage by hitting **OK**.

[L66] 

[L67] ![grant access to external storage ok](https://support.optisigns.com/hc/article_attachments/36321849036051)

[L68] 

[L69] Hit **Install**.

[L70] 

[L71] ![install options apkinstaller](https://support.optisigns.com/hc/article_attachments/36321880545683)

[L72] 

[L73] Here, find your External Storage device on the path menu.

[L74] 

[L75] ![usb memory](https://support.optisigns.com/hc/article_attachments/36321880549139)

[L76] 

[L77] Select the **OptiSigns APK** file. You should see the below screen. Hit **Install.**

[L78] 

[L79] ![optisigns app install confirmation](https://support.optisigns.com/hc/article_attachments/36321849045139)

[L80] 

[L81] If everything goes well, you should see this screen:

[L82] 

[L83] ![optisigns app successful install](https://support.optisigns.com/hc/article_attachments/36321880557715)

[L84] 

[L85] Now all that's left is [pairing your screen to your account](https://support.optisigns.com/hc/en-us/articles/360016374813-Set-up-add-a-screen).

[L89] ## Installing APK via Web Browser

[L90] 

[L91] These steps should be followed if you have **access to a Web Browser, but not the Google Play store** on your Android device.

[L92] 

[L93] ### Download OptiSigns APK and Install It

[L94] 

[L95] To begin, open your web browser (Chrome will be used in this example).

[L96] 

[L97] Type in the following URL: <https://www.optisigns.com/download>

[L98] 

[L99] Then, click **Go.**

[L100] 

[L101] ![download latest version of optisigns app via web browser](https://support.optisigns.com/hc/article_attachments/1500019255902)

[L102] 

[L103] Click **The APK** button in the Android Operation System on [our site.](https://www.optisigns.com/download)

[L104] 

[L105] ![download optisigns apk web browser](https://support.optisigns.com/hc/article_attachments/29651805277459)

[L106] 

[L107] Click **Download** to download the OptiSigns APK file

[L108] 

[L109] ![confirm download](https://support.optisigns.com/hc/article_attachments/1500019255962)

[L110] 

[L111] Click on the **Options** button (three-period button) in Chrome.

[L112] 

[L113] ![find downloads](https://support.optisigns.com/hc/article_attachments/1500019255602)

[L114] 

[L115] Click **Downloads**

[L116] 

[L117] ![options button chrome](https://support.optisigns.com/hc/article_attachments/1500019255622)

[L118] 

[L119] Click the **OptiSigns APK** file, then hit **Install.**

[L120] 

[L121] ![downloads button](https://support.optisigns.com/hc/article_attachments/1500019431121)

[L122] 

[L123] Click the **Done** button.

[L124] 

[L125] ![install optisigns apk](https://support.optisigns.com/hc/article_attachments/1500019431141)
