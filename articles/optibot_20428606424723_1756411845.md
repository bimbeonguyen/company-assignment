[L1] # How to Use Coordinate-Based Positioning and Sizing of Elements in OptiSigns' Designer

[L2] 

[L3] If you want to craft visually appealing and professional designs, achieving pixel-perfect layouts and precise element positioning is essential.

[L4] 

[L5] In this article, we'll guide you on using coordinates to accurately place and size elements on your canvas.

[L6] 

[L7] ¬†

[L8] 

[L9] # **Ready to get started? Then, let's jump in:**

[L10] 

[L11] * Go to **Designer** App by going to your Files/Assets page ‚Üí Apps

[L12] 

[L13] ![](https://support.optisigns.com/hc/article_attachments/20428249099923)

[L14] 

[L15] * Click on any text or image to access the menu options on the top.

[L16] 

[L17] ![](https://support.optisigns.com/hc/article_attachments/20428309851923)

[L18] 

[L19] Click on '**Position**' on the top right hand corner

[L20] 

[L21] By clicking on the '**Position**', this will display the **Position options** on the right hand side of your canvas.

[L22] 

[L23] ![](https://support.optisigns.com/hc/article_attachments/20428297986195)

[L24] 

[L25] ¬†

[L26] 

[L27] Under the **Advanced** section, you will see the following options:

[L28] 

[L29] ![](https://support.optisigns.com/hc/article_attachments/20428510772371)

[L30] 

[L31] * **Left** - Input a numeric value to specify the horizontal position of the selected element within your canvas.

[L32] 	+ By entering a numerical value, you can specify a numerical value to determine how far the element is positioned from the left side of the canvas.

[L33] * **Top** - Input a numeric value to specify the vertical position of the selected element within your canvas.

[L34] 	+ By entering a numerical value, you can specify a numerical value to determine how far the element is positioned from the top edge of the canvas.

[L35] * **Width** - This option lets you define the horizontal size, or width, of the selected element.

[L36] 	+ You can input a numerical value to specify how wide the element should be.

[L37] * **Height** - This option allows you to set the vertical size of the selected element.

[L38] 	+ By entering a numerical value, you determine how tall the element should be.

[L39] * **Maintain Aspect Ratio link (for images or graphical elements only)**¬†- This feature is a convenient tool for maintaining the original proportions of the selected element while adjusting its width and height.

[L40] 	+ When activated (with the link set to "Vertical," as shown above), changing one dimension (width or height) will automatically adjust the other dimension to preserve the aspect ratio.

[L41] 	+ This ensures that the element doesn't become distorted when resized.
