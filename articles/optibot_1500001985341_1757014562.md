[L1] # How to Use the Local Folder App in OptiSigns

[L2] 

[L3] ### In this article, we'll explain how to use the Local Folder app in OptiSigns to let you access locally stored files on your device to display.

[L4] 

[L5] * [How to Set Up the Local Folder App](#Setup)

[L6] 	+ [For Windows](#Windows)

[L7] 	+ [For Mac/Linux/Ubuntu/Raspberry Pi](#Mac)

[L8] 	+ [For the OptiSigns Android Stick](#Stick)

[L9] 	+ [For the OptiSigns Pro Player](#ProPlayer)

[L10] 

[L11] Sometimes, due to network limitations or security reasons, you'll have content you'd rather keep locally at your device. The Local Folder app allows you to access content stored locally.

[L12] 

[L13] Major use cases for this app include:

[L14] 

[L15] * Sensitive information that should not be transferred via Internet

[L16] * Limited network availability locations

[L17] * You want to use or mix in contents from USB thumb drives

[L18] 

[L19] OptiSigns player is smart enough to scan for changes in these folders. When files are added or removed, it's handled automatically.

[L20] 

[L21] You can also use USB thumb drives. OptiSigns detects drives plugged in and scan the folder to play again as long as it's given the correct path.

[L29] ## **How to Set Up the Local Folder App**

[L30] 

[L31] First, you will need to have your screens set up and paired. For more information on how to do that, click┬Ā[here](https://www.optisigns.com/blog/how-to-set-up-digital-signs-with-optisigns-and-amazon-fire-tv).

[L32] 

[L33] Then log on to our portal:┬Ā<http://app.optisigns.com/>

[L34] 

[L35] Go to **Files/Assets** and click on **"Apps"**:

[L36] 

[L37] ![optisigns portal with arrows pointing to Files/Assets tab and Apps button](https://support.optisigns.com/hc/article_attachments/21882432568595)

[L38] Find the┬Ā**Local Folder** app:

[L39] ![apps screen in optisigns portal with arrow pointing at local folder app](https://support.optisigns.com/hc/article_attachments/360102953613)

[L40] ┬Ā

[L41] Enter the path to your local folder (explained in more detail below) and other settings:

[L42] ![local folder app menu](https://support.optisigns.com/hc/article_attachments/360100790274)

[L43] 

[L44] * **Name** - Name of your Local Folder app. This will be displayed on your asset list, and will **not**┬Ābe displayed on your screens.

[L45] * **Local Folder** - Path to the folder on your device. We will provide detailed instructions per platform later in this article.

[L46] * **Transition Effect** - Transition effect between your images and videos in the folder.

[L47] * **Transition Speed** - How fast you want your transitions to animate.

[L48] * **Duration for Image** - How long each images should show on your screen, default is 10 seconds.

[L49] * **Max Video Duration** - Default is 0, which means the app will play each video till the end. You can set it to some other values to prevent too long videos. For example if you set this value to 60 seconds. If a video is longer than 60s, it will get cut off at 60s to play next item. If a video is less than 60s, it will play the duration of the video.

[L50] * **Scale Image** - select how the player should handle if image's resolution is less than your screen's resolution

[L51] * **Scale Video** - select how the player should handle if the video's resolution is less than your screen's resolution

[L52] * **Shuffle** - select if you want to shuffle play the content in the folder (Default order is by filename alphabetically)

[L53] 

[L54] Click **Save**.

[L55] 

[L56] You now can assign the Local Folder app to your device. It can also be placed in a Playlist or Schedule.

[L57] 

[L58] ### For Windows:

[L59] 

[L60] If using a Windows device to display your content, the path to your Local Folder will be similar to C:\ D:\Media Folder etc. Find the path by looking at the folder's Properties, like so:

[L61] 

[L62] ![windows folder highlighting folder path](https://support.optisigns.com/hc/article_attachments/33807447378707)

[L63] 

[L64] You'll then use this path to fill in the Local Folder value within the Local Folder App:

[L65] 

[L66] ![local folder app in optisigns with windows value](https://support.optisigns.com/hc/article_attachments/33807447390355)

[L67] 

[L68] This will vary depending on the location of the folder on your drive.

[L69] 

[L70] ### For Mac/Linux/Ubuntu/Raspberry Pi:

[L71] 

[L72] If using a Mac, Linux, or Ubuntu device, the path to your folder will be similar to /media/path/folder.

[L73] 

[L74] For Mac users, find this using **Get Info**. It will look similar to this:

[L75] 

[L76] ![mac folder highlighting folder path](https://support.optisigns.com/hc/article_attachments/33807447394067)

[L77] 

[L78] This comes out to /Users/{username}/Desktop as a path filename.

[L79] 

[L80] In Linux/Ubuntu, you can find this information in **Properties.**

[L81] 

[L82] ![ubuntu folder highlighting folder path](https://support.optisigns.com/hc/article_attachments/33807672236819)

[L83] 

[L84] The path in this example would be /home/optisigns .

[L85] 

[L86] For Raspberry Pi, the path can also be found in **Properties**.

[L87] 

[L88] ![raspberry pi folder highlighting folder path](https://support.optisigns.com/hc/article_attachments/33807431768339)

[L89] 

[L90] In this example, that's /home/pi.

[L91] 

[L92] You'll then use this path to fill in the Local Folder value within the Local Folder App:

[L93] 

[L94] ![local folder app with mac/linux/ubuntu/pi value](https://support.optisigns.com/hc/article_attachments/33807431781907)

[L95] 

[L96] This will vary depending on the location of the folder on your drive. However, it's important to note that Mac, Linux, Ubuntu, and Raspberry Pi paths all have the same format.

[L118] ### For OptiSigns Pro Digital Signage Player

[L119] 

[L120] The OptiSigns Pro Player is an Ubuntu enabled device, so it runs the same way. You'll have a slightly different path depending on whether you're using a USB device or MicroSD card as your external storage, or if you have multiple USB drives plugged into the player.

[L121] 

[L122] The path for a single USB device when plugged into the OptiSigns Pro Player will be:

[L123] 

[L124] ```

[L125] /home/optisigns/external/sdb1

[L126] ```

[L127] 

[L128] If your device has a label, it will be:

[L129] 

[L130] ```

[L131] /home/optisigns/external/{label}-sdb1

[L132] ```

[L133] 

[L134] Where {label} is the name/label of the USB drive.

[L135] 

[L136] Should you plug in a second USB drive, the path will be:

[L137] 

[L138] ```

[L139] /home/optisigns/external/sdc1

[L140] ```

[L141] 

[L142] Similarly, a third USB drive will have a path of:

[L143] 

[L144] ```

[L145] /home/optisigns/external/sdd1

[L146] ```

[L147] 

[L148] Again, remember to add the label to the path if your drive has one.

[L149] 

[L150] Lastly, if you're using a MicroSD card, your path will be:

[L151] 

[L152] ```

[L153]  /home/optisigns/external/{label}-mmcblk0p1

[L154] ```

[L155] 

[L156] Where {label} is the name/label of the MicroSD card.

[L157] 

[L158] Then, use this path to fill in the Local Folder value within the Local Folder App:

[L159] 

[L160] ![local folder app optisigns with pro player value](https://support.optisigns.com/hc/article_attachments/33807447426963)
