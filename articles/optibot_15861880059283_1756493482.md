[L1] # Silent Installation process for OptiSigns Player on Windows

[L2] 

[L3] By default, when you install the OptiSigns Windows player on your Windows device, it will be installed on your default user profile.   

[L4]   

[L5] If you want to do the silent install in another folder instead of the default user profile. This will be helpful for looking for mass provisioning on Windows and deploying with a prebuilt Windows image.

[L6] 

[L7] You can follow this article.

[L8] 

[L9]  

[L10] 

[L11] 1. Search the **Command Prompt** and **Run as administrator**

[L12] 

[L13] ![](https://support.optisigns.com/hc/article_attachments/15861776979603)

[L14] 

[L15] 2. Move the OptiSigns exe file to the folder. 

[L16] 

[L17] 3. Run or go to your folder

[L18] 

[L19] ```

[L20] cd c:\temp

[L21] ```

[L22] 

[L23] or

[L24] 

[L25] ```

[L26] cd c:\[Your Folder Name]

[L27] ```

[L28] 

[L29] 4. Run or install the OptiSigns in your folder

[L30] 

[L31] ```

[L32] "OptiSigns Digital Signage Setup 5.6.4" /S /D="c:\temp\optisigns"

[L33] ```

[L34] 

[L35] or

[L36] 

[L37] ```

[L38] "OptiSigns Digital Signage Setup 5.6.4" /S /D="c:\[Your Folder Name]\optisigns"

[L39] ```

[L40] 

[L41] ![](https://support.optisigns.com/hc/article_attachments/15912711216275)

[L42] 

[L43]  

[L44] 

[L45] Then it will install in that folder.

[L46] 

[L47]
