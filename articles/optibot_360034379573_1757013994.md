[L12] 

[L13] The OptiSigns Player app can be installed on any Linux machine and used as a player for your digital signs.

[L14] 

[L15] OptiSigns app is packaged as AppImage and will run on all major Linux Distribution: Ubuntu, Debian, CentOS, Red Hat, Arch Linux, etc.

[L16]

[L20]

[L28] ## Installing OptiSigns on any Linux Machine

[L29] 

[L30] Once you've got the latest AppImage downloaded on your system, locate it wherever it's been downloaded.

[L31] 

[L32] ![optisigns appimage linux](https://support.optisigns.com/hc/article_attachments/37230635291283)

[L33] Right click on the AppImage and select **Properties**.

[L34] ![appimage properties](https://support.optisigns.com/hc/article_attachments/37230646236179)

[L35] 

[L36] In the Properties box, ensure the Permissions are set to **Read & Write** and the "Executable as Program" slider is **On**.

[L37] 

[L38] ![optisigns appimage properties menu](https://support.optisigns.com/hc/article_attachments/37230646238995)

[L39] 

[L40] We're not done. Before we can launch the application, we'll need to configure the proper libraries.

[L41] 

[L42] Now return to where your AppImage is stored and double click it. If you are running certain distributions that does not come with libfuse2, you will need [an additional step](#24.04).

[L43] You'll be asked to confirm Install, hit **Yes**.

[L44] ![install linux appimage window](https://support.optisigns.com/hc/article_attachments/37230646240915)

[L45] 

[L46] The app will run in full screen mode and generate a pairing code for you to pair with the [app.optisigns.com](http://app.optisigns.com/) portal. If you move the mouse around, you will see the top 3 buttons to **resize**, **open** **side bar menu,** or **close** the app.

[L47] 

[L48] ![optisigns player pair code](https://static.wixstatic.com/media/e48f7f_3ee0de008d8b4f038c0de10250e80749~mv2.png/v1/fill/w_925,h_522,al_c,q_90,usm_0.66_1.00_0.01/e48f7f_3ee0de008d8b4f038c0de10250e80749~mv2.webp)

[L49] 

[L50] On the side menu, you can set Orientation, etc. The app will have the AutoStart and Fullscreen on Startup features checked as default. However, more setup is needed if you'd like the app to AutoStart.

[L51] 

[L52] ![optisigns player side menu](https://static.wixstatic.com/media/e48f7f_2fc697a5f1224304b2a6b23835a38355~mv2.png/v1/fill/w_925,h_518,al_c,q_90,usm_0.66_1.00_0.01/e48f7f_2fc697a5f1224304b2a6b23835a38355~mv2.webp)

[L56] ## Installing OptiSigns on Ubuntu v. 24.04 or Later

[L57] 

[L58] Ubuntu v. 24.04 introduced some changes which will require some additional steps to get OptiSigns working properly.

[L59] 

[L60] First, follow [all the instructions above](https://support.optisigns.com/hc/en-us/articles/37162616928659#24.04). Next, you'll need to download the libfuse2 driver.

[L61] 

[L62] ### Install libfuse2 Library to Support AppImage

[L63] 

[L64] To do this, open your **Terminal** and type:

[L65] 

[L66] ```

[L67] sudo apt install libfuse2

[L68] ```

[L69] 

[L70] ![linux terminal libfuse2 install command](https://support.optisigns.com/hc/article_attachments/37230646245011)

[L71] 

[L72] The necessary libraries will automatically install, and now the OptiSigns AppImage can be launched.

[L73] 

[L74] Now, there are a few optional things you can do to further optimize OptiSigns on Ubuntu 24.04 or later, including:

[L75] 

[L76] * Enabling AutoStart

[L77] * Enabling Video Hardware Acceleration via MPV Player

[L78] 

[L79] These are completely optional, but helpful for your experience.

[L80] 

[L81] ### Enabling AutoStart

[L82] 

[L83] To enable AutoStart, we will need to alter the startup configuration on Ubuntu. To do this, we'll be setting up a shell script to run independently. This can be easily accomplished in around two minutes via Startup Applications Preferences.

[L84] 

[L85] Open up **Startup Application Preferences** and you should see something like this:

[L86] 

[L87] ![linux startup programs](https://support.optisigns.com/hc/article_attachments/37230635312147)

[L88] 

[L89] The issue here is that the OptiSigns Digital Signage will not work after repeated startups. To get it working, hit **Add.**

[L90] 

[L91] ![linux startup add program](https://support.optisigns.com/hc/article_attachments/37230635315091)

[L92] 

[L93] Name the file whatever you please and input any Comment you want. The **Command:** line is what's important here: put the same path as the basic OptiSigns autostart program, but add:

[L94] 

[L95] ```

[L96] --no-sandbox

[L97] ```

[L98] 

[L99] To the end of the command.

[L100] 

[L101] Once this is done, your player will continually autostart with each boot. You can delete the old OptiSigns startup script if you wish.

[L102] 

[L103] ### Enabling Video Hardware Acceleration via MPV Player

[L104] 

[L105] In certain situations, Linux running Ubuntu 24.04 or later can struggle to play multiple or very high quality videos using the OptiSigns player.

[L106] 

[L107] In these situations, we recommend a very simple set of commands that should fix the issue.

[L108] 

[L109] First, if you're **using an Intel processor,** you'll need to download the proper drivers. Type this into the Terminal:

[L110] 

[L111] ```

[L112] sudo apt install intel-media-va-driver-non-free

[L113] ```

[L114] 

[L115] Next, you'll need to install MPV Player with this command:

[L116] 

[L117] ```

[L118] sudo apt install mpv

[L119] ```

[L120] 

[L121] Last, you'll want to hide the taskbar and top bar in the player. You'll need to input this command in order to get started:

[L122] 

[L123] ```

[L124] sudo apt install gnome-shell-extension-manager

[L125] ```

[L126] 

[L127] From here, head to **Settings.** Go to **Ubuntu Desktop**, and make sure **Auto-hide the Dock** is switched to ON.

[L128] 

[L129] ![linux settings auto hide dock](https://support.optisigns.com/hc/article_attachments/37230646253459)

[L130] 

[L131] Next, open up the Extension Manager. Here, click **Browse**, then search for the "Hide Top Bar" extension. Click **Install**.

[L132] 

[L133] ![linux extension manager hide top bar](https://support.optisigns.com/hc/article_attachments/37230893391763)

[L134] 

[L135] Now your video hardware acceleration should be enabled.
