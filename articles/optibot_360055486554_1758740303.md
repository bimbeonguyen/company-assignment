[L139] ### Additional Commands

[L140] 

[L141] Hitting the arrow near the buttons brings up a list of additional commands.

[L142] 

[L143] **![additional commands menu](https://support.optisigns.com/hc/article_attachments/43590071784723)**

[L144] 

[L145] * **Refresh & Relaunch:** Refresh and relaunch your screen. This is useful if your screen is frozen or is not functioning as intended, but you don't want to reboot the device.

[L146] * **Send HDMI CEC** - Use HDMI CEC commands to remotely turn on/off TV. Please make sure HDMI CEC support is enabled on the TV.

[L147] * **Send RS232:**This is used to utilize RS232 Commands to remotely turn your TV On/Off.

[L148] * **Get used storage/free storage:** Gives information on your used and free device storage.

[L149] * **Install OptiSigns Versions:** Lets you remotely install different versions of OptiSigns. If you downgrade from the current version, please make sure to turn off Auto Updates, otherwise it will automatically update to the current version (Android Stick Player only).

[L150] * **Disable Auto Update:** This will allow you to remotely disable automatic updates on your OptiSigns player. This is useful when you downgrade from the current version (Android Stick Player only).

[L151] * **Install APK:** Remotely install APK with a direct link/URL to the APK file (Android Stick Player only).

[L187] ## For Multiple OptiSigns Players

[L188] 

[L189] ItŌĆÖs possible to send basic remote commands to individual players, segments of OptiSigns players (defined via [tagging](https://support.optisigns.com/hc/en-us/articles/38062664690195-Tagging-in-OptiSigns)), or all your OptiSigns players simultaneously. [L201] ### How to Execute Remote Commands

[L202] 

[L203] To submit the command to manage the devices remotely, you will need to go to the remote command execution console. Click **Screens**, then the **More Options (Three Dots)**

[L204] 

[L205] Screens -> More Options Icon -> Execute Remote Commands

[L206] 

[L207] ![more options execute remote commands](https://support.optisigns.com/hc/article_attachments/43590071795219)

[L208] 

[L209] The Execute Remote Commands console can be broken down into three sections:

[L210] 

[L211] * **Section 1** - Where you specify the target of the commands.

[L212] * **Section 2** - The command you want to execute.

[L213] * **Section 3** - The execution result and history.

[L214] 

[L215] ![execute remote commands steps](https://support.optisigns.com/hc/article_attachments/43590102885907)

[L216] 

[L217] For section 1, there are 2 types of targets you can choose from:

[L218] 

[L219] * **Screens** - You can select the screen name here, it can be one screen or multiple screens.

[L220] * **Tags** - Utilizing tags, you can execute the command on a group of devices. In the below example, the command will be submitted to all devices tagged as Windows or Raspberry Pi.

[L221] 

[L222] ![target and tags execute remote commands](https://support.optisigns.com/hc/article_attachments/43590071798035)

[L223] 

[L224] For section 2, you can enter the command you want to execute in the text box. The command needs to be OS-specific, depending on the OS your device is running on (Android for Android Stick, Linux for OptiSigns Pro / ProMax players), you will need to build the scripts accordingly. Once you have your commands ready, just click the submit button. The command will be pushed to the devices for execution.

[L225] 

[L226] Other than the free-form scripts, there are some common commands built into the platform, to achieve these functions, you will just need to select it from the drop-down menu. Depending on the type of device OS, the command will be submitted accordingly.

[L227] 

[L228] ![refresh and relaunch](https://support.optisigns.com/hc/article_attachments/43590071799571)

[L229] 

[L230] After a command is executed, it will appear in section 3: the command history.

[L237] ### Controlling Remote Commands Received by Players

[L238] 

[L239] In some cases, you may not want some of your devices to be controlled remotely for any reason. The ability to receive remote commands can be toggled on the device.

[L240] 

[L241] Go to the side menu on the OptiSigns player, click **Advanced Options**, and you'll find the toggle for **Execute Remote Commands**. Toggling this off will cause the device to reject remote commands.

[L242]
