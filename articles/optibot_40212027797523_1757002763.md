[L30] ## How to Set Up Encryption on an OptiSigns Player

[L31] 

[L32] To get started, attach a keyboard to your OptiSigns Player. When booting up, rapidly press the **Up Arrow** to gain access to the Restore mode menu. It should look like this:

[L33] 

[L34] ![restore mode menu optisigns player](https://support.optisigns.com/hc/article_attachments/40212059658387 "ApplicationFrameHost_DOBZcmN4V9.png")

[L35] 

[L36] Here, hit Factory Reset. YouŌĆÖll be asked whether or not to enable encrypted disks. Type **Yes** and hit Enter.

[L37] 

[L38] ![enabling encryption restore mode](https://support.optisigns.com/hc/article_attachments/40212059660051 "ApplicationFrameHost_BNOVLclemr.png")

[L39] 

[L40] It will ask you to create a password. Create one, then complete the Factory Reset.

[L41] 

[L42]   | **! VERY IMPORTANT !** |

[L43] | The password you set up will be required to be input every time you wish to recover data from your drive. Make sure to save this password in a safe place that youŌĆÖll have access to, or else you will not be able to retrieve data from the device. |

[L44] 

[L45] If you bring up Restore mode again, you should see the **Encrypted Boot** option there. This means you have successfully encrypted the OptiSigns Player drive. From now on, it will be booted with the encryption in place.
