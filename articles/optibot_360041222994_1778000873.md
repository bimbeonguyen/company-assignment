[L10] The CountUp (or Count Up) app is a simple way to display a customizable count up clock for any occasion.┬Ā

[L14] The clock can display any title you like, and when it goes off, can display any message you like. Here, we'll show how to create one. Let's get started.

[L26] ## Creating a CountUp Asset

[L27] 

[L28] From the OptiSigns portal, go to **Files/Assets** **ŌåÆ Apps:**

[L29] 

[L30] ![](https://support.optisigns.com/hc/article_attachments/46720888202259)

[L31] 

[L32] On the Apps menu, select **CountUp:**

[L33] 

[L34] ![](https://support.optisigns.com/hc/article_attachments/46720907996563)

[L35] 

[L36] The below screen should open up:

[L37] 

[L38] ![](https://support.optisigns.com/hc/article_attachments/46720888203283)

[L39] 

[L40] These are the options, and what they do. Any edits made to these options should be automatically reflected in the **Preview** pane on the right. The **Preview** can be displayed in either **Landscape** or **Portrait** format.

[L41] 

[L42] * **Name:** The name of the CountUp asset. This is for saving the asset in OptiSigns, and the text entered here will not appear on the asset itself.

[L43] * **Text:** The text to be displayed on the CountUp asset while it counts down. By default, this says "OptiSigns Timer," and it is in this area this message will be displayed.

[L44] * **End Time:** When selected, will bring up a calendar:

[L45] 

[L46] ![](https://support.optisigns.com/hc/article_attachments/46720888203539)

[L47] 

[L48] * This allows you to choose the precise date and time for the countup to be completed.

[L49] * **Message:** The message to be displayed at the end of the countup.

[L50] * **Timer Style:** Lets you choose the type of Timer to display. Choose between **Flip**, **Basic**, **Ring**, and **Slide:**  

[L51] ![](https://support.optisigns.com/hc/article_attachments/46720907997587)  

[L52] Specific differences between these can be seen in the Preview panel.

[L53] * **Days Only:** Check this box to only show how many days are left on your countup timer.

[L54] * **Theme:** Lets you change between a number of basic themes. Scroll down for more. Their specifics can be seen in the Preview panel.

[L73] ### **Advanced Settings**

[L74] 

[L75] Expandable panel. Click this for even more options:

[L78] * **Recurring:** When checked, enables recurring events to be repeated at selected intervals. Opens up more options:  

[L79] ![](https://support.optisigns.com/hc/article_attachments/46720888206995)

[L80] 	+ **Repeat Every:** Lets you choose how often the countup repeats. You can adjust this number, and choose from **Day, Week, Month,** or **Year**. When **Week, Month,** or **Year** is chosen, it will show what days it will **Repeat On**.  

[L81] 	  

[L82] 	**Week:**  

[L83] 	![](https://support.optisigns.com/hc/article_attachments/46720888207251)  

[L84] 	**Month:**

[L86] 	**Year:**  

[L87] 	**![](https://support.optisigns.com/hc/article_attachments/46720888207635)**

[L88] 	+ **Ends:** Lets you choose when a recurring countup timer will end. This can be **Never**, **On**, or **After**.

[L89] 		- **Never:** The countup timer will repeat as per above instructions and will not stop recurring until asset is changed.

[L90] 		- **On:** Allows you to set a specific date at which time the countup timer will stop recurring.

[L91] 		- **After:** Allows you to set a specific number of recurrences after which the timer will stop recurring.
