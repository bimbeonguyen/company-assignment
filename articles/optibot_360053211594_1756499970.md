[L1] # Set OptiSigns as ScreenSaver on Windows

[L2] 

[L3] Sometimes, you would want to set OptiSigns to run if the PC is idle for some time.  

[L4] This is a good use case for Kiosk and Public use computers.

[L5] 

[L6] In this guide, we will walk you through end to end process to set OptiSigns app as Screen Saver on Windows.

[L7] 

[L8] ### How to Set OptiSigns as Screen Saver on Windows.

[L9] 

[L10] First open OptiSigns side menu, click Advanced, and turn on "Screen Saver Mode".

[L11] 

[L12] When this mode is on, OptiSigns will be closed on any mouse move, click, or keyboard pressed.

[L13] 

[L14] ![mceclip0.png](https://support.optisigns.com/hc/article_attachments/4403429024787)

[L15] 

[L16] Then use Window's task scheduler to run OptiSigns when the PC is idle.

[L17] 

[L18] Here's the steps, on Windows click Start, search for "Task Scheduler" & open it.

[L19] 

[L20] Task Scheduler -> Create Task -> "Trigger" tab -> New -> "Begin the task:" -> "On Idle"

[L21] 

[L22] Next, go to the "Actions" tab and set the action to whatever it is you want to run.

[L23] 

[L24] * Search for "**Task Scheduler**" & open it

[L25] 

[L26] ![Task_Scheduel_1.png](https://support.optisigns.com/hc/article_attachments/360091166373)

[L27] 

[L28] * **Create Task** in your┬ĀTask Scheduler

[L29] 

[L30] ![Task_Scheduel_2.png](https://support.optisigns.com/hc/article_attachments/360091166393)

[L31] 

[L32] * Set **Configure** for your device

[L33] 

[L34] ![Task_Scheduel_3.png](https://support.optisigns.com/hc/article_attachments/360091166413)

[L35] 

[L36] * Go to **Triggers** tap, and click "**New ...**"

[L37] 

[L38] ![Task_Scheduel_4.png](https://support.optisigns.com/hc/article_attachments/360091166433)

[L39] 

[L40] * Set Begin the task: "**On idle**"

[L41] 

[L42] ![Task_Scheduel_5.png](https://support.optisigns.com/hc/article_attachments/360091166453)

[L43] 

[L44] * Go to **Actions** tap, and click "**New ...**"

[L45] 

[L46] ![Task_Scheduel_6.png](https://support.optisigns.com/hc/article_attachments/360091166473)

[L47] 

[L48] * Set Action: "**Start a program**", and "**Browse**" OptiSigns app's location.

[L49] 

[L50] ![Task_Scheduel_7.png](https://support.optisigns.com/hc/article_attachments/360089015334)

[L51] 

[L52] * Go to **Condition** tap, and set the condition for your computer.

[L53] 

[L54] ![Task_Scheduel_8.png](https://support.optisigns.com/hc/article_attachments/360089015354)

[L55] 

[L56] * Go to **Setting** tap, and set "**Run a new instance in parallel**" (The Task Scheduler service will run the new instance of the task in parallel with the instance that is already running.)

[L57] 

[L58] ![Task_Scheduel_9.png](https://support.optisigns.com/hc/article_attachments/360089015394)

[L59] 

[L60] * After that, you finish your task setting. You can click "**Run**" to start this task.

[L61] 

[L62] ![Task_Scheduel_10.png](https://support.optisigns.com/hc/article_attachments/360089015494)

[L63] 

[L64] * Your status of task will show "**Running**".

[L65] 

[L66] ![Task_Scheduel_11.png](https://support.optisigns.com/hc/article_attachments/360091166533)

[L67] 

[L68] * Close OptisSigns app

[L69] 

[L70] Once the computer is idle, your screen will run OptiSigns app.┬Ā

[L71] 

[L72] You're ready to go.

[L73] 

[L74] ┬Ā

[L75] 

[L76] **IMPORTANT:**

[L77] 

[L78] Because OptiSigns is closed on any mouse move, click or key press when Screen Saver mode is turned on.

[L79] 

[L80] If you want to control OptiSigns again, press Ctrl + Alt + A or Ctrl + Alt + S, this will turn off Screen Saver mode, and you can interact with OptiSigns.
