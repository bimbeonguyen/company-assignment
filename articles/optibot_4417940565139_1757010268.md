[L1] # How to display Microsoft Teams on OptiSigns

[L54] ### Why does using Teams require giving OptiSigns Admin permission to use?

[L55] 

[L56] OptiSigns uses Microsoft APIs for integration. In order for our integrations to work, the integration has to be approved by an administrator. This is the same across all integrations using Microsoft APIs.

[L57] 

[L58] This administrator access is only needed for first time access. Once the OptiSigns app is approved for use, other users can use OptiSigns directly.
